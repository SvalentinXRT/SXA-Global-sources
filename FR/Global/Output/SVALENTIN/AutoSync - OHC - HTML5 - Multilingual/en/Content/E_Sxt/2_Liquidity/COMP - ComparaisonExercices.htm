Comparing Exercises

This function enables to compare the treasury budgets for several budget exercises.

You can thus compare:

  • Several versions of a budget exercise
  • Budget exercises of different entity groups
  • Budget exercises for different years (monthly or quarterly comparison for this option)
Accessing Compare Exercises page

In the Modules Tasks pane, select the Compare Exercises option from the Management folder of Liquidity.

  • If no filter has been set up to compare exercises, the filter configuration page is displayed.

The filters for Compare Exercises function enable you to define the criteria and display settings for the function.

Refer to the topic below for more information on setting up filters for the Compare Exercises function.

  • If a filter has already been set up, the Compare Exercises page is displayed.

This page is similar to the budget sheet in Planning mode, except for the fact that, instead of a comparison between operational budget and treasury budget of a single exercise, this a comparison between treasury budgets of different exercises.

  • The left pane consists of two parts.
  • In the upper part, the tree structure of the budget exercises presents the elements selected in the applied display filter.

On the first level of the tree structure, two nodes appear:

  • "Exercises selection 1 " node displays the first group of exercises selected in the filter.
  • "Exercises selection 2 " node displays the second group of exercises selected in the filter.

Both groups are then compared in the main pane, as described below. The amounts of the budget transactions are cumulated within each group depending on the selected periodic division.

This tree structure is not editable. To select other exercises, you need to modify the filter for Compare Exercises function. Refer to the topic below for more information on setting up filters for the Compare Exercises function.
  • In the lower part, the filter configuration is summarized. This is the same as the summarized filter configuration of the budget sheet.

For more information on quick filters, see the Display Filters topic.

  • The right pane displays your compared budgets sheet for which you defined the active filter and the display preferences.

The Treasury Budgets for the two compared exercises groups can both be displayed on the same page, depending on the settings defined for the display filter. Each table presents amounts by period and by budget code. Budget codes are classified by category.

From this page, you can view the detailed amounts and transactions' interrelations. You may also export the data from the main pane to Excel.

For more information on:

  • setting up filters for exercise comparison, see the topic hereafter,
  • the display preferences, see the Display Preferences topic,
  • setting up the pane display, see the Panes topic,
  • the display of amounts details, see the Details of a Transaction Amount topic (this function works as the budget sheet one),
  • viewing dependencies between transactions, see the Dependency Relationships between Transactions topic (this function works as the budget sheet one),
  • the export to Excel, click here.

For more information on the display setup, see the Display Preferences topic.
