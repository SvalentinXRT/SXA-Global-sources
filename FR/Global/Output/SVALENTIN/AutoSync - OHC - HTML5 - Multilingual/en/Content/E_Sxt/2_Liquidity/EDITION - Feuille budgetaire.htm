Budget Sheet Report

This report enables you to generate a sheet presenting your budget according to customized criteria.

You can set up the layout and the selection criteria using the report filters.

Accessing Report Page of the Budget Sheet

In the Modules Tasks pane, select the Budget sheet option in the Report folder of the Liquidity module.

  • If no filter has been set up for this report, the filter configuration page is displayed. See the Report Filters topic for more information on setting up filters for this report.
  • If a filter has already been set up for this report, the report page opens up, presenting the budget sheet generated with the default filter.

The layout of the report table depends on the settings of the default filter.

View Report Description

The body of the report shows one after the other the revisions that you have selected in the filter, with, for each revision:

  • a one-line title displaying the revision identifier, as well as the codes for the company, exercise period, type of budget sheet (operational or treasury), exercise status and revision.
  • the revision budget in a table.

The budget table and the entry budget sheet are similar.

  • Amounts are grouped by budget code and period.
  • The table lines group transaction amounts by budget code. Budget codes are classified by category. Category totals are also presented for each category.
  • The table columns group transaction amounts by period. Periods are determined by the standard length of a period defined in the report filter: day, week, month, etc.

For each period, depending on the settings defined in the filter, three columns can be displayed: they specify the amount saved in the treasury budget, the amount saved in the Cash module and the discrepancy between both amounts, in amount or percentage according to the configuration of the report filter.

Displaying the consolidated view of Budgets for a Company Grouping

The report filters enable you to set up a consolidated view of budgets for a company belonging to a same grouping.

The followig process describes the selection of a budget sheet designer in the report filter and the settings for the consolidated view.

See the Report Filters topic for more information on the filter configuration for this report.

  1. In the Filters toolbar of the report page, click the Modify option.

The modification page for the report filter settings, is displayed.

  1. In the setup page for the report filter, select a budget sheet designer as follows:

In the Designer area of the Companies tab, click the button to display the list of the set up designers. In the contextual window, double-click the line to select it.

The code and description of the selected designer are displayed in the Designer area of the Companies tab.

  1. Click on the Consolidation tab.

This tab enables you to set up a report which groups budgets at the level of the companies groupings on several exercises.

  1. In the Exercises area, click the Select button to display the list of the existing exercises for the selected companies and periods. In the contextual window, enable the options in front of the exercises to display in the report, then click the button .

The codes of the selected exercises are displayed:

  • in the Exercises area with the designer code associated with the exercise,
  • in the Revision area before the name of the last available version (initial, revised, or final versions). You can select an older version than the last one.
  1. To select a previous version, click the revision code and the line, in the contextual window, to select it.

The selected version is displayed in the Revision column.

  1. Click the following buttons, as needed:
  • Save to save the setup without leaving the current page
  • OK to save the setup and display again the report Budget Sheet
  • Cancel to display again the report Budget Sheet without saving the setup
Showing Forecasts only

The following process describes the setup for the forecasts display in the report filter.

See the Report Filters topic for more information on the filter configuration for this report.

  1. In the Filters toolbar of the report page, click the Modify option.

The modification page for the report filter settings, is displayed.

  1. In the setup page for the report filter, click the View tab.

  1. In the BS views area, select the Budget Planning option.
  1. Click OK to save the setting and display again the Budget Sheet report.
Viewing Forecasts, Actual and Discrepancy

The following process describes the setup for the forecasts analysis, for the flows of the Cash module and for the discrepancies in the report filters.

See the Report Filters topic for more information on the filter configuration for this report.

  1. In the Filters toolbar of the report page, click the Modify option. 

The modification page for the report filter settings, is displayed.

  1. In the setup page for the report filter, click the View tab.

  1. In the BS views area, select the Budget Analysis option.

Enable the Totals Only option to display only the subtotals, the totals of the category, the opening balance, etc.

  1. Set up the display options of the analysis as decsribed in the table below.
