Exercises

Setting up a budget exercise consists in selecting:

  • an Organization (setup described in the Organizational Structure topic),
  • a Currency,
  • Budget Codes and Macros (setup described in the Budget Codes and Macros topic),
  • an Exercise Period (setup described in the topic below).

Then, the following elements must be defined for the exercise: Opening balances for the Operational Budget and the Treasury Budget, the distribution of amounts for each unit (if units have been defined for the selected company), the Breakdown Rules for flows of the Cash module to Budget Transactions, as well as the creation rules for Cash Flows in Cash based on Budget Transactions.

After creating all potential correspondences between Cash Flow Types and Budget Codes, you can analyze the discrepancies between Forecasts and Actuals.

Breakdown Rules for Cash Flows

These rules define how Budget Transactions are created based on Cash Flows. To set them up, you need to associate one or more budget codes with a cash flow type, and with a distribution percentage for the original flow amount.

Creation Rules for Cash Flows

These rules define how Cash Flows are created based on Budget Transactions. To set them up, you need to associate one or more cash flow types and bank accounts with a budget code, then each bank account and each flow type with a distribution percentage of the original flow amount.
