Cash Ledger

For each transaction, you can view the details of the cash flows generated in the Cash module or a recap in the Payment module, from the homepage in the Analysis section.

You may also view the detailed ledger in the Cash module.

You can view the main information on the cash flows linked to the transactions for the transfer types managed in Payment (Payment Requests, Domestic, International transfers, etc.).

Viewing a Ledger for a type of Transfer in Payment
  1. From the homepage of the Payment module, in the Analysis menu, select the transfer type for which you want to see the ledger, for example Domestic Transfer Ledger.

  1. If you do not want to filter the domestic transfers displayed in the ledger, click directly OK without completing any field.

The following screen is displayed:

The transactions associated with the selected transaction are displayed in a first table.

The second table displays general information on the currency and the total amount of the transactions linked to the selected transfer.

Viewing the Cash Ledger report in Cash
  1. In the accordion menu of the Modules Tasks pane, select the Cash module to display the Cash menu.
  2. In the Report menu of the Cash module, select the Cash Ledger report.

If a filter has already been set up for this report, the report page opens up, presenting the Cash Ledger in the main pane.

If no filter has been set up for this report, the creation page for Filters is displayed. See the Cash documentation for more information on how to create Filters for the Cash Ledger Report.

The header presents the type and date of the report, the filter name, as well as the filtering criteria defined in the current filter:

  • periods of the value date, transaction date or update date for the flows,
  • entities,
  • display unit for the amounts,
  • display of decimals,
  • flows status (initial, forecasted, actual),
  • view of the amounts' details or totals.
