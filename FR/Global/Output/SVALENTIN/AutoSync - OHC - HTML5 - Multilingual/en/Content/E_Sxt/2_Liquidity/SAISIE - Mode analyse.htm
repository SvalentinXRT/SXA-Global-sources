Budget Analysis

The Budget Analysis mode enables you to view and defer the discrepancies between the Treasury Budget and the flows saved in Cash.

Analysis enables you to roll Non-actual Forecasts.

The budget sheet is displayed in analysis mode when you select this mode in the budget sheet filter.

The discrepancy between Forecasts and Actuals is null over the mixed budget period if, since the last actuals import, no cash flow was created nor modified in Cash for the dates, company and budget codes involved in the budget exercise. For more information on the mixed budget, see the Importing Actuals procedure.

For more information on:

  • the setup options of the Analysis Mode (display of the budget and of the discrepancy with the flows of the Cash module), see the Budget Sheet Entry and Analysis topic.
  • the options of the display filters (companies, division into periods, etc.), see the Display Filters topic,
  • the rolling of non-actual forecasts, see the Rolling Non-actual Forecasts topic.
  • the activation of the analysis mode, click here.

Showing or Hiding Discrepancies
  1. Dans la feuille budgétaire, cliquez sur le bouton Modifier dans la barre d'outils Filtres.
    La page de paramétrage du filtre sélectionné s'affiche.
  1. Cliquez sur l'onglet Affichage.

  1. Dans la zone Affichage FB, activez la case d'option Analyse de budget.
    Les champs de la zone suivante, spécifique à l'analyse de budget, sont activés.
  2. Dans la zone Affichage de l'écart, activez la case d'option :
  • Ne pas afficher pour ne pas afficher l'écart (option indisponible si la case d'option Écart est activée dans la zone Type d'analyse),
  • Montant pour afficher le montant de l'écart,
  • Pourcentage pour afficher le pourcentage de l'écart.
  1. Si vous avez choisi d'afficher le budget de trésorerie et les flux enregistrés dans le module Cash dans deux feuilles distinctes (options Budget + Trésorerie et Grille double activées dans les zones Type d'analyse et Mode analytique), activez l'un des boutons d'option suivants dans la zone Écart pour grille double :
  • Grille B pour afficher l'écart dans le budget de trésorerie (grille en haut),
  • Grille T pour afficher l'écart dans la feuille des flux enregistrés dans le module Cash (grille en bas).

Une fois vos modifications terminées, dans la barre d'actions située en haut de la page, cliquez sur le bouton :

  • Enregistrer pour enregistrer vos modifications sans quitter la page,
  • OK pour enregistrer vos modifications et revenir à la page précédente,
  • Annuler pour annuler vos modifications et revenir à la page précédente.

Pour plus d'informations sur :

  • les options de configuration du mode analyse (présentation du budget et de l'écart avec les flux du module Cash), consultez la rubrique Budget Sheet Entry and Analysis,
  • les options des filtres d'affichage (entités, découpage périodique, etc.), consultez la rubrique Display Filters.
Showing or Hiding Treasury Budget or Flows saved in the Cash Module
  1. In the budget sheet, click the Change button in the Filters toolbar.
    The setup page for the selected filter is displayed.

  1. Click the View tab.

  1. In the BS views area, enable the Budget Analysis option.
    The fields of the area (specific to the budget analysis) are enabled.
  2. In the Type of Analysis area, enable the option:
  • Budget + Treasury to display the Treasury Budget and the flows saved in the Cash module,
  • Budget to display only the Treasury Budget,
  • Treasury to display only the flows saved in the Cash module,
  • Discrepancy to show only discrepancies and hide the Treasury Budget and the flows saved in the Cash module.
  1. If you chose to view the Treasury Budget and the flows saved in the Cash module, enable one of the following options in the Analysis Mode area:
  • Single Grid to display a single sheet (the Treasury Budget amounts are above the flows saved in the Cash module),
  • Double Grid to display two sheets with the Treasury Budget and the flows saved in the Cash module in two different sheets.

  1. In the Display Discrepancy area, enable the option:
  • Not show not to display the discrepancy (this option is disabled when the Discrepancy option is not selected in the Type of Analysis area),
  • Amount to display the discrepancy amount,
  • Percentage to display the discrepancy percentage.
  1. If you selected the double grid and the display of the discrepancy (enabled Amount or Percentage options in the Display Discrepancy area), enable one of the following options in the Discrepancy for Double Grid area:
  • Display in B grid to display the discrepancy in the Treasury Budget,
  • Display in T grid to display the discrepancy in the sheet of the flows saved in the Cash module.

Une fois vos modifications terminées, dans la barre d'actions située en haut de la page, cliquez sur le bouton :

  • Enregistrer pour enregistrer vos modifications sans quitter la page,
  • OK pour enregistrer vos modifications et revenir à la page précédente,
  • Annuler pour annuler vos modifications et revenir à la page précédente.

For more information on:

  • the setup options of the Analysis Mode, see the Budget Sheet Entry and Analysis topic,
  • the options of the display filters, see the Display Filters topic.
