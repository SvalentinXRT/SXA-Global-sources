Budget Entries Ledger

This report enables you to generate a list of the transactions posted in the budget according to selection criteria that you specify.

The selection criteria are defined in the report filters: for example you can specify budget codes to include or exclude.

Accessing Report Page of the Budget Entries Ledger

In the Modules Tasks pane, select the Budget Entries Ledger option in the Report folder of the Liquidity module.

If a filter has already been set up for this report, the report page opens up, presenting the budget entries generated with the default filter.

If no filter has been set up for this report, the filter configuration page is displayed. See the Report Filters topic for more information on setting up filters for this report.

View Report Description

The header displays the type and date of the report, the filter name and the budget exercise period.

The table below describes the column of the report table.

Column Description
Exercise Budget exercise code associated with the transaction
Budget Code Budget code of the transaction
Trans. Date Date of the transaction
Amount Amount of the transaction
Currency Currency of the transaction
Description Description of the transaction
T

Type of Budget Sheet linked to the Transaction:

"OBS" is for Operational Budget Sheet

"TBS" is for Treasury Budget Sheet
EXS Exercise status associated with the transaction:
INI corresponds to the initial version.
REV corresponds to a revised version.
CLO corresponds to the final version of the closed exercise.
O Origin of the transaction amount:
ST corresponds to a transaction calculated with a formula.
BS corresponds to a transaction amount that was entered.
S Transaction status:
E corresponds to an enabled transaction.
D corresponds to a disabled transaction.
