Sage XRT Advanced 2.0

Bank Area

In Sage XRT Advanced 2.0, the Bank Area module is designed to centralize the imported bank entries. Further analysis and statistical tools will continue to be integrated to the following versions.

You can access Bank Area from a new submenu in the Setup menu. In this module, imported statements and entries are available.

During successful imports, statements are automatically posted into the Cash and Reconciliation modules. In case of rejections, usually due to setup errors, and once errors have been corrected, statements can be manually posted.

The Bank File Entries imported onto the platform can be accessed via the menus: Administration – Bank File Entry.

The imported Bank File Statements as well as their statuses can be accessed via the menus: Administration – Bank File Statement.

Market Data

The module is designed to centralize the market data imported from an external source or entered manually.

The Market Data function enabling the manual input or import of currency rates and quotation places can be accessed from a new submenu in the Setup menu.

Import File: JSON Format

The Import (JSON) function is now also available for the following entities: Currencies and Countries. The goal is to make these basic entities automatically up and running.

JSON import can be initiated from the menus: Action – Export – Import File – Import (JSON).

Action: Comments

The Comments action which enables you to add notes, is now available for all entities. It is displayed as of you select a line.

To display this action into the grid, add the Comments/Entries/ID column to the current view.

Default Dashboards

Three new dashboards are provided: Reconciliation, Setup and Bank Delegations.

Improved Grid

Quick Selection

Two options for multiline selection are now available:

  • place your cursor on a line and from it, drag the lines to select,
  • place your cursor on a line, press the Shift key and select the selection-ending line. All the elements between the two lines are added to your selection.

Scrollable List

The Paging disabled and Load on scroll options must be activated within the view.

The page numbers at the bottom of the screen disappear and the lines are now displayed as a scrollable list.

Auto-adjusting Columns

The Auto-adjust Column Width and Horizontal scroller & frozen columns options must be activated within the view.

Filters on Formatted Amounts

The formatted amounts consist of three elements: Amount, Currency and Direction. You can now sort by one of these elements through the quick filter.

Improved Reject Management

In the G/L Reject Management, data can be modified field by field before another import.
