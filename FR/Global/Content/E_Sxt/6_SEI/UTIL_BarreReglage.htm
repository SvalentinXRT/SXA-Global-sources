Barre de réglage

Le volet Barre de réglage permet d'accéder aux options les plus fréquemment utilisées.

Pour ouvrir la Barre de réglage, cliquez sur le bouton situé en-haut à droite de la fenêtre.

  • Déconnecter : Ce bouton permet de se déconnecter et de fermer l'application .
  • Ouvrir lien externe : Ce lien peut être utilisé pour accéder directement à une vue ou pour intégrer la vue dans une autre application.
L'utilisation de cette option consomme une licence Mobile.
  • Ouvrir lien externe web : Ce lien peut être utilisé pour accéder directement à une vue ou pour intégrer la vue dans une autre application.
L'utilisation de cette option consomme une licence Web.
  • Exporter : Ce sous-menu permet d'exporter des vues au format PDF, Excel ou HTML.
  • Langue : Ce sous-menu permet de modifier la langue utilisée pour la présentation des données d'une vue. L'anglais est sélectionné par défaut.
  • Environnement : La source des données peut être modifiée en sélectionnant cette option.
  • Fermer : Ce bouton permet de refermer le volet Barre de réglage.

Selon le contexte d'utilisation, certaines options de ne sont pas disponibles. Ces options peuvent être désactivées pour les raisons suivantes :

  • l'utilisateur en cours n'a pas les droits d'accès à l'option
  • l’option ne peut pas s'appliquer (par exemple, lorsqu'aucune vue n'est ouverte, l'option Exporter les données n'est pas proposée).
