Configuration de la structure du fichier à importer

La configuration de la structure d'un fichier externe consiste à définir :

  • la nature des données présentes dans le fichier,
  • l'emplacement des données dans le fichier,
  • la correspondance des données avec la base de données au moyen d'une table de correspondance, lorsque les codes diffèrent entre le fichier et la base de données.
Accéder à la page de paramétrage des structures de fichier

Dans le volet , sélectionnez l'option Structure d'importation de budget dans le dossier du module .

La page de paramétrage des structures de fichiers s'affiche avec la liste des structures paramétrées sous forme de tableau. Le tableau présente le code et le libellé de chaque structure, ainsi que le type de fonction concernée, le format de date attendu, la longueur type d'une transaction en caractères et l'existence ou non d'équivalences de valeurs dans une table de correspondance.

Pour des informations sur :

  • la réorganisation de l'affichage des lignes du tableau, cliquez ici,
  • la recherche de texte dans le tableau, cliquez ici.
  • la création d'une ligne, consultez la rubrique ci-après.
  • la modification de la structure et des équivalences dans le tableau, cliquez sur le bouton Modifier la structure ou sur le bouton Modifier les correspondances. Vous accédez aux mêmes pages qu'en création.
  • la suppression d'une ligne dans le tableau, cliquez ici.

  1. Dans la page de paramétrage de structures de fichier, cliquez sur le bouton Créer.

  1. Cliquez sur le bouton  Paramétrer les tables de correspondance pour continuer le paramétrage.

La page de paramétrage des tables de correspondance s'affiche.

  • Si les codes d'entités, de devises et de natures budgétaires dans le fichier sont identiques à ceux de la base de données, cliquez sur le bouton  Tester la structure pour sauter cette étape de paramétrage et passer directement au test de la structure.
  • Pour sauter l'étape de test de la structure de fichiers et valider directement le paramétrage, cliquez sur le bouton  Valider.
  • Pour annuler le paramétrage en cours et réafficher la liste des structures de fichiers paramétrées, cliquez sur le bouton  Annuler.


Cette page comporte trois onglets, permettant chacun de configurer la table de correspondance concernée : Entités, Codes budgétaires ou Devises.

Chaque onglet présente un tableau à trois colonnes.

  • La première colonne, intitulée N°, comporte un numéro identifiant chaque correspondance.
  • La deuxième colonne, intitulée Zone interne, comporte les codes enregistrés dans la base de données de .
  • La troisième colonne, intitulée Zone externe, comporte les codes enregistrés dans le fichier.
  1. Cliquez sur un onglet, cliquez sur le bouton Ajouter pour ajouter une ligne de valeurs équivalentes dans l'onglet.
  2. Dans la ligne qui s'affiche dans le tableau, cliquez sur le bouton dans la colonne Zone interne pour sélectionner le code interne à la base de données. Dans la colonne Zone externe, saisissez le code attendu dans le fichier provenant du système externe.
  3. Cliquez sur le bouton Ajouter pour ajouter une nouvelle ligne ou sur le bouton Supprimer pour supprimer la ligne sélectionnée.

Répétez les étapes 7 à 8 autant de fois que nécessaire pour compléter la table de correspondance, puis procédez de même dans les autres onglets.

  1. Cliquez sur le bouton  Tester la structure pour tester le paramétrage sur un fichier.

La page de test de la structure s'affiche.

  1. Dans la zone Fichier, cliquez sur le bouton pour afficher l'explorateur du module et sélectionner un fichier à tester.

Une fenêtre contextuelle s'affiche.

  1. Dans l'explorateur, sélectionnez le dossier où se trouve le fichier.
    Le chemin s'affiche dans la zone Chemin sélectionné et les fichiers contenus dans le dossier sont répertoriés dans le tableau au bas de la fenêtre.
  2. Double-cliquez sur un nom de fichier pour le sélectionner ou cliquez dessus, puis cliquez sur le bouton pour valider votre sélection.

La page de l'Assistant est réaffichée et le chemin du fichier est affiché dans le champ Fichier.

  1. Cliquez sur le bouton Tester.

Les transactions trouvées sont affichées dans le tableau avec pour chacune le code de l'entité, le code de la nature budgétaire, le montant de la transaction, sa devise, la date de l'opération et le descriptif de la transaction.

Le nombre de transactions trouvées dans le fichier s'affiche en bas du tableau.

  1. Vérifiez que l'analyse du fichier correspond au résultat attendu en termes de nombre de transactions et de valeurs dans les champs. Corrigez la structure ou le fichier au besoin.
  2. Une fois que le test a fourni les résultats attendus, cliquez sur le bouton  Valider pour valider et enregistrer la structure.

La page de paramétrage des structures de fichiers est réaffichée avec, en bas de la liste des structures paramétrées, la nouvelle structure enregistrée.

Vous pouvez passer à l'étape suivante, le paramétrage des modalités d'importation.

 Dupliquer une structure de fichier

Vous pouvez créer une structure en dupliquant les paramètres d'une structure existante.

  1. Dans la page de paramétrage des structures, cliquez sur la structure à dupliquer, puis cliquez sur le bouton Dupliquer dans la barre d'actions.
  2. La page de création de structure s'affiche, avec les paramètres de la structure dupliquée. Un numéro identifiant est créé automatiquement pour la structure par incrémentation.

  3. Dans la zone Libellé, saisissez le libellé de la nouvelle structure.

Vous pouvez modifier les paramètres de la structure dupliquée.

Pour plus d'informations sur ces champs, consultez la rubrique précédente.

 Modifier les correspondances de valeurs entre fichier et base de données

Vous pouvez enrichir ou modifier en permanence les correspondances de valeurs entre fichier et base de données.

  1. Dans la page de paramétrage des structures, cliquez sur la structure à modifier, puis cliquez sur le bouton Modifier les correspondances dans la barre d'actions.
  2. La page de définition des tables de correspondance s'affiche, avec les correspondances paramétrées pour la structure sélectionnée.

Pour plus d'informations sur les champs de cette page, consultez la rubrique décrivant la création de structure de fichier à importer.
