Enregistrement, diffusion, impression

Enregistrement

Le module vous permet d'enregistrer l'édition en cours de consultation dans de nombreux formats de fichier et à des emplacements spécifiés.

  1. Dans la barre d'actions de la page des éditions, cliquez sur le bouton Exporter édition.
  2. Dans la boîte de dialogue qui s'affiche, sélectionnez le format final du fichier et l'emplacement du fichier généré, puis cliquez sur le bouton OK.

Les éditions peuvent être enregistrées dans différents formats : CSV, HTML, ODBC, PDF, REC, RPT, RTF, TTX, TXT, XLS, XML.

 Envoi par courrier électronique

Le module vous permet de générer un fichier au format HTML ou PDF et de l'envoyer par courrier électronique.

  1. Dans la barre d'actions de la page des éditions, cliquez sur le bouton Notifier.
  2. Dans la boîte de dialogue qui s'affiche, sélectionnez le format final pour la diffusion, puis cliquez sur le bouton OK.

Une fenêtre contextuelle de progression s'affiche pendant le traitement de la tâche.

Votre application de messagerie démarre automatiquement. Un message est créé avec, en pièce jointe, le fichier généré à partir de l'édition, au format que vous avez sélectionné.

 Impression
  1. Dans la barre d'actions de la page des éditions, cliquez sur le bouton Paramètres de l'imprimante.

La fenêtre de configuration de l'impression s'affiche.

Renseignez :

  • la taille et la localisation du papier dans l'imprimante,
  • l'orientation du papier (paysage ou portrait),
  • les dimensions des marges d'impression.
  1. Sélectionnez une imprimante et cliquez sur OK.
  2. Dans la barre d'actions de la page des éditions, cliquez sur le bouton Imprimer.

La fenêtre de configuration avancée de votre imprimante s'affiche, vous permettant de lancer l'impression.

Pour plus d'informations sur la configuration de l'impression dans cette fenêtre, consultez la documentation de votre système d'exploitation.
