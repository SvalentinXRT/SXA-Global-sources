Devises et cours budget

Il existe deux types de cours de devises dans  :

  • le cours marché, qui est le cours effectif d'une devise, utilisé par le module pour les flux,
  • le cours budget, qui est le cours prévisionnel utilisé par le module pour les transactions budgétaires.

Il peut exister une différence entre le cours budget et le cours marché. Cette différence est visible dans les écarts que vous affichez dans la page de saisie du budget (en mode analyse) ou dans l'édition de feuille budgétaire.

Le paramétrage des devises est une étape nécessaire préalable à la création des feuilles budgétaires. Lors du paramétrage d'une entité (consultez la rubrique Structure organisationnelle), vous devez sélectionner une devise.

Le cours budget doit être paramétré si vous travaillez dans plusieurs devises. Par exemple, lorsqu'un groupe est composé d'entités dont le budget est établi dans différentes devises, les budgets des différentes entités peuvent être regroupés dans une vue consolidée au niveau groupe et dans une devise unique.

La valeur du cours budget est définie sur une base mensuelle.

La devise est accessible via le module et le cours budget via le module .

Définir une devise
  1. Accédez à la page de paramétrage des devises :
    1. Dans le volet , sélectionnez le module dans le menu accordéon.
      Les tâches du module s'affichent dans ce volet.
    2. Sélectionnez l'option Devises dans le dossier de ce module.

    La page de paramétrage des devises s'affiche avec la liste des devises sous forme de tableau.

    Le tableau présente le code et le libellé de chaque devise en deuxième et troisième colonnes.

  2. Créez une devise :
  1. Cliquez sur le bouton Créer.
    L'onglet Généralités de la page de création des devises s'affiche.

  1. Sélectionnez un code dans la liste déroulante Code de la zone Identification.
  1. Cliquez sur le bouton :
  • Appliquer pour enregistrer le paramétrage de la devise et en créer une autre,
  • OK pour enregistrer le paramétrage de la devise et réafficher la liste des devises paramétrées,
  • Annuler pour annuler le paramétrage de la devise.

L'étape suivante, décrite dans la procédure ci-après, consiste à définir un cours budget entre la nouvelle devise et la devise de référence.

 Définir un cours budget
  1. Dans le volet Tâches de modules, cliquez sur le module dans le menu accordéon. Les tâches du module s'affichent dans ce volet.
  2. Sélectionnez l'option Cours budget dans le dossier .

La page de paramétrage des cours budget s'affiche avec la liste des cours prévisionnels paramétrés sous forme de tableau.

Les codes des couples de devises sont affichés avec leurs valeurs prévisionnelles pour chaque mois de l'année en cours.

  1. Pour sélectionner un couple de devises, activez la case à cocher à gauche d'une ligne et cliquez sur le bouton Modifier.
    La page de détail du cours budget s'affiche.

La zone Identification affiche les informations d'identification de la devise dont vous allez définir le cours par rapport à la devise de référence. Cette zone n'est pas modifiable.

  1. Dans la zone Années en bas à droite, sélectionnez l'année à paramétrer.
    Les cours budget pour chaque mois de l'année sélectionnée s'affichent dans la zone Cours de l'année.
  2. Saisissez les valeurs du cours budget pour chaque mois.
  1. Si vous souhaitez copier la valeur d'un mois vers les mois suivants de l'année, placez votre curseur dans la valeur à copier et cliquez sur le bouton Dupliquer.

Les valeurs des mois suivants sont écrasées et mises à jour avec la valeur copiée.

  1. Une fois votre cours budget défini pour le couple de devises sélectionné, cliquez sur le bouton OK pour enregistrer votre saisie ou sur le bouton Annuler pour annuler le paramétrage en cours.
