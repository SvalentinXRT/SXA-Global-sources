Analyse du budget

Le mode analyse vous permet de consulter et de reporter les écarts entre le budget de trésorerie et les flux enregistrés dans le module .

Grâce à l'analyse, vous pouvez reporter les prévisions non réalisées.

La feuille budgétaire s'affiche en mode analyse lorsque vous sélectionnez le mode analyse dans le filtre de feuille budgétaire.

L'écart affiché entre le prévisionnel et le réalisé est nul sur la période de budget mixte si, depuis la dernière importation du réalisé, aucun flux de trésorerie n'a été créé ou modifié dans le module pour les dates, la société et les natures budgétaires concernées par l'exercice budgétaire. Pour plus d'informations sur le budget mixte, consultez la procédure Importer le réalisé.

Pour plus d'informations sur :

  • les options de configuration du mode analyse (présentation du budget et de l'écart avec les flux du module ), consultez la rubrique Activer le mode analyse,
  • les options des filtres d'affichage (entités, découpage périodique, etc.), consultez la rubrique Filtre avancé,
  • le report des écarts, consultez la rubrique Report des prévisions non réalisées,
  • l'activation du mode analyse, cliquez ici.

Afficher ou masquer l'écart Afficher ou masquer le budget de trésorerie ou les flux enregistrés dans le module Cash
  1. Dans la feuille budgétaire, cliquez sur le bouton Modifier dans la barre d'outils Filtres.
    La page de paramétrage du filtre sélectionné s'affiche.

  1. Cliquez sur l'onglet Affichage.

  1. Dans la zone Affichage FB, activez la case d'option Analyse de budget.
    Les champs de la zone suivante, spécifique à l'analyse de budget, sont activés.
  2. Dans la zone Type d'analyse, activez la case d'option :
  • Budget + Trésorerie pour afficher le budget de trésorerie et les flux enregistrés dans le module ,
  • Budget pour afficher uniquement le budget de trésorerie,
  • Trésorerie pour afficher uniquement les flux enregistrés dans le module ,
  • Écart pour afficher uniquement l'écart et masquer le budget de trésorerie et les flux enregistrés dans le module .
  1. Si vous avez choisi d'afficher le budget de trésorerie et les flux enregistrés dans le module , activez l'un des boutons d'option suivants dans la zone Mode analytique :
  • Grille simple pour afficher une seule et même feuille (montants du budget de trésorerie au-dessus des flux enregistrés dans le module ),
  • Grille double pour afficher le budget de trésorerie et les flux enregistrés dans le module dans deux feuilles distinctes.

Afficher un exemple de grille double

  1. Dans la zone Affichage de l'écart, activez la case d'option :
  • Ne pas afficher pour ne pas afficher l'écart (option désactivée si la case d'option Écart est activée dans la zone Type d'analyse),
  • Montant pour afficher le montant de l'écart,
  • Pourcentage pour afficher le pourcentage de l'écart.
  1. Si vous avez choisi la grille double et l'affichage de l'écart (option Montant ou Pourcentage activée dans les zones Afficher l'écart), activez l'un des boutons d'option suivants dans la zone Écart pour grille double :
  • Grille B pour afficher l'écart dans le budget de trésorerie,
  • Grille T pour afficher l'écart dans la feuille affichant les flux enregistrés dans le module .
