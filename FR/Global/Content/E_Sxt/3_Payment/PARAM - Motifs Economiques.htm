Motifs économiques

Le motif économique est uniquement utilisé pour les Déclarations Banque de France. Il s'agit de la raison économique justifiant un transfert hors frontières et permettant à la banque centrale de calculer la balance des paiements.

En création des virement, la saisie du motif est libre, sauf pour les ordres associés au formats AFB320 et EDI91, lesquels rendent obligatoire cette information si le donneur d'ordre ou le bénéficiaire est non résident. Il convient donc de paramétrer les différents motifs possibles, pour ultérieurement, pouvoir les sélectionner à partir d'une liste existante.

La liste des motifs les plus courants vous est livrée avec le produit (voir cette liste en annexe), mais vous avez la possibilité de la compléter et de l'adapter à vos besoins.

Alimenter la table des motifs économiques standard
  1. Pour accéder à cette fonctionnalité, dans le volet , sélectionnez l'option Eléments de paiement dans le dossier du module .

La page de configuration des éléments de paiement s'affiche. Cliquez sur le lien Motifs économiques.

La liste des motifs disponibles s'affiche à l'écran.

  1. Cliquez sur le bouton Initialisation automatique pour visualiser la boîte de dialogue Sélection.
  2. Optez pour l'une des deux solutions suivantes pour procéder à la sélection :
  • Cliquez sur le bouton Tout, si vous avez l'intention d'utiliser tous les motifs prédéfinis. Une coche s'affiche dans les cases de sélection. Si certains motifs de la liste ne vous intéressent pas, décochez-les en cliquant sur les cases de sélection correspondantes.
  • Cochez les cases de sélection des motifs qui vous intéressent lorsqu'ils sont peu nombreux.
  1. Cliquez sur Ok pour valider l'opération, puis confirmez-la en choisissant le bouton Oui dans la boîte de message qui s'affiche.

Le contenu de la fenêtre Motifs économiques a changé.

Le bouton Créer du menu contextuel Motifs économiques permet d'afficher la boîte de dialogue Motifs économiques - Créer. Les motifs économiques étant imposés par la Banque de France et étant livrés en standard avec le produit, vous aurez peu l'occasion d'avoir à en créer de nouveaux. Néanmoins, vous pouvez par exemple personnaliser les libellés si vous ne les trouvez pas suffisamment explicites.

Le code d'un motif économique n'est pas modifiable. Vous devez supprimer ce motif et le recréer avec un nouveau code.
Créer les motifs économiques
  1. Pour accéder à cette fonctionnalité, dans le volet , sélectionnez l'option Eléments de paiement dans le dossier du module .

La page de configuration des éléments de paiement s'affiche. Cliquez sur le lien Motifs économiques.

La liste des motifs disponibles s'affiche à l'écran.

  1. Cliquez sur le bouton Créer pour visualiser la boîte de dialogue Motifs économiques - Créer.

  1. Renseigner le champ obligatoire Code. Le code doit avoir une taille maximum de trois caractères alphanumériques.
  2. Saisissez une description du motif dans le champ Libellé. Votre libellé est limité à un nombre de 35 caractères.
  3. Une fois les champs renseignés, cliquez sur Appliquer pour valider votre opération et poursuivre vos créations de motifs ou cliquez sur OK pour enregistrer le nouveau motif et refermer la boîte de dialogue.
 Modifier un motif économique
  1. Sélectionnez l'option Eléments de paiement dans le dossier du module . Cliquez sur le lien Motifs économiques.
  2. Sélectionnez le motif à modifier dans la liste.
  3. Cliquez sur Modifier. La boîte de dialogue Motifs économiques - Modifier s'affiche.
  4. Modifiez le contenu du champ Libellé, puis cliquez sur OK pour sauvegarder votre opération.
 Supprimer un motif économique
  1. Sélectionnez l'option Eléments de paiement dans le dossier du module . Cliquez sur le lien Motifs économiques.
  2. Sélectionnez le motif à supprimer dans la liste.
  3. Cliquez sur Supprimer. La boîte de dialogue Motifs économiques - Supprimer s'affiche.
  4. Confirmez votre opération en cliquant sur Oui dans la boîte de message qui s'affiche.
