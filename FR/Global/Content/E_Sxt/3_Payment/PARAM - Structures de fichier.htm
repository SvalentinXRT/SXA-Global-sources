Structure des fichiers échangés

Préalablement à l'importation et à l'exportation de virements, vous devez créer les structures de fichiers correspondantes dans le module .

Une structure de fichier doit exister pour chaque type de fichier importé ou exporté afin que l'intégration dans la base de données de ou l'extraction depuis cette dernière puisse être effectuée.

La configuration de la structure d'un fichier consiste à définir :

  • la nature des données présentes dans le fichier,
  • l'emplacement des données dans le fichier,
  • la correspondance des données avec la base de données au moyen d'une table de correspondance, lorsque la codification des informations diffère entre le fichier et la base de données.
