Configuration de l'affichage

Les filtres de feuilles budgétaires vous permettent :

  • de déterminer des critères de sélection des données (périodes, entités, etc.),
  • de basculer entre les modes saisie et analyse de budget,
  • de déterminer la présentation de l'écart entre le budget et les flux enregistrés dans le module , en mode analyse de budget.

Vous devez avoir configuré au moins un filtre de feuille budgétaire avant de pouvoir saisir votre budget.

Les préférences d'affichage vous permettent de déterminer la présentation graphique des données affichées, comme les polices de caractères et les couleurs. Ces préférences permettent notamment de mettre en valeur les montants par type (transactions budgétaires, flux de trésorerie, écarts positifs, écarts négatifs...).

La vue consolidée vous permet de consulter, en lecture seule, une vue groupée des budgets de plusieurs entités pour une même période.
