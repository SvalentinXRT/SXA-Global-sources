Tables de répartition

Le module vous permet de paramétrer des tables de répartition, utilisées lors de la création des flux de trésorerie dans le module .

Lorsque vous lancez la création des flux de trésorerie à partir du budget de trésorerie, une transaction budgétaire peut donner lieu, en fonction du paramétrage effectué dans l'exercice budgétaire, à la création de plusieurs flux de trésorerie sur différents comptes bancaires, chaque flux ayant un montant, une nature de flux et une date d'opération distincte.

Ce sont les tables de répartition qui, au moyen d'un calendrier et d'un système de pondération, vous permettent de définir les pourcentages de répartition des montants sur des jours choisis dans le mois.

Les tables de répartition doivent être créées avant l'exercice budgétaire car c'est lors du paramétrage de l'exercice budgétaire que vous sélectionnez la table de répartition à appliquer.

Accéder à la page de paramétrage des tables de répartition

Dans le volet , cliquez sur Table de répartition dans le dossier du module .

La page de paramétrage des tables de répartition s'affiche avec la liste des tables paramétrées sous forme de tableau.

Le tableau présente le code et le libellé de chaque table.

En modification, seul le libellé d'une période d'exercice est modifiable.

 Créer une table de répartition
  1. Dans la barre d'outils Actions de la page de paramétrage des tables de répartition, cliquez sur le bouton Créer.
    La page de création de table de répartition s'affiche.

  1. Saisissez un code et un libellé pour la table de répartition dans les champs Code et Libellé. Le code ne pourra pas être modifié ultérieurement.
  2. Pour associer la table à une entité, dans la zone Entité, sélectionnez une entité comme suit :
  1. Cliquez sur le bouton pour afficher la liste des entités.
  2. Dans la fenêtre contextuelle qui s'affiche, double-cliquez sur une ligne pour la sélectionner.

Le code et le libellé de l'entité s'affichent dans le champ Entité

  1. Dans la zone Table, saisissez les coefficients de pondération correspondant aux règles de répartition à définir sur un mois, comme expliqué ci-dessous.
  • La première colonne du tableau affiche les semaines du mois. Six semaines sont affichées par défaut, mais vous pouvez définir vos coefficients sur le nombre de semaines correspondant à votre calendrier.
  • La deuxième colonne du tableau vous permet de définir un coefficient global pour chaque semaine.
  • Les autres colonnes du tableau vous permettent de définir un coefficient individuel pour chaque jour.

Chaque coefficient journalier est ensuite multiplié par le coefficient de la semaine correspondante.
