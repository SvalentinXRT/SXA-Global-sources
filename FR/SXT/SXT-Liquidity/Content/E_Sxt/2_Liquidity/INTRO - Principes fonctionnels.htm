Principes fonctionnels généraux

Dans le module , vous gérez deux types de feuille budgétaire.

Feuille budgétaire opérationnelle

Elle représente vos prévisions basées sur la production, hors taxes et avec la date prévisionnelle des opérations. Vous effectuez initialement vos prévisions dans cette feuille.

Vous pouvez constituer des prévisions par importation de données comptables.

L'importation est décrite dans la rubrique Importation de données comptables.

 Feuille budgétaire de trésorerie

Elle représente l'état de votre trésorerie à un instant t, toutes taxes comprises et avec la date prévue de l'encaissement ou du décaissement.

La feuille budgétaire de trésorerie est obtenue à partir de la feuille budgétaire opérationnelle par application de formules de calcul que vous définissez pour chaque nature budgétaire. Sont ainsi calculées la date de l'encaissement ou du décaissement, les taxes sur la valeur ajoutée, les charges sociales, etc.

Vous pouvez également remplacer les prévisions passées dans la feuille budgétaire de trésorerie par le réalisé (flux de trésorerie créés dans le module ). Vous travaillez alors sur un budget dit mixte.

L'écart entre le prévisionnel et le réalisé est calculé avec les flux de trésorerie enregistrés dans le module .

Vous pouvez ainsi réviser votre budget à partir de cette analyse et reporter les prévisions non réalisées.

Vous pouvez également analyser votre budget en comparant différents exercices budgétaires : comparaison entre différentes versions d'un même exercice, entre différents groupes d'entités ou entre différentes années.

Un budget peut être révisé indéfiniment jusqu'à la clôture de l'exercice. Les versions successives d'un budget sont conservées pour consultation.

La gestion des feuilles budgétaires est décrite dans la rubrique Saisie et analyse du budget

Préalablement à la saisie du budget, vous devez effectuer l'intégralité du paramétrage décrit dans la rubrique Paramétrage.

Ce paramétrage vous permet de définir l'environnement de votre activité et la structure de la feuille (natures budgétaires, règles de calcul, présentation des informations...).

Une fois le budget finalisé, vous pouvez générer automatiquement les flux de trésorerie correspondants dans le module à partir des transactions du budget de trésorerie.

Enfin, vous pouvez générer des éditions à partir des feuilles budgétaires, enregistrer ces éditions dans une multitude de formats et les diffuser.

La gestion des éditions est décrite dans la rubrique Éditions.
