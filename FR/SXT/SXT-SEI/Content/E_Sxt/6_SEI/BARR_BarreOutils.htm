Barre d'outils

  • Déconnexion : Ce bouton permet de quitter l’application et de revenir sur l’écran d’accueil.
  • Lien externe d’une vue : sélectionnez cette option pour afficher la vue en cours dans un nouvel onglet du navigateur Web. Un lien pouvant être utilisé pour accéder à la vue sans utiliser apparaît alors dans la barre d’adresse. Ce lien permet à d’autres applications d'accéder à cette vue.
L'utilisation de cette option consomme une licence Mobile.

  • Lien externe d’une vue Web : sélectionnez cette option pour afficher la vue en cours dans un nouvel onglet du navigateur Web. Un lien pouvant être utilisé pour accéder à la vue sans utiliser apparaît alors dans la barre d’adresse. Ce lien permet à d’autres applications d'accéder à cette vue.
L'utilisation de cette option consomme une licence Web.

  • Exporter : Ce sous-menu permet d'exporter la vue en cours au format PDF, Excel ou HTML. Un message s'affiche permettant de sélectionner un emplacement pour le nouveau fichier exporté.

  • Sélection de la langue : peut fournir un support multilingue. Le nom des processus, des vues et des champs peut être disponible dans plusieurs langues, à condition que l'administrateur ait activé cette option. La fonction de sélection de la langue permet de basculer entre les différentes langues activées.
  • Sélection de l’environnement : Ce menu permet de sélectionner l'environnement des données d'exploitation de l'application. Le nom de l’environnement par défaut est affiché en en-tête. Les environnements sont configurés par l’administrateur et peuvent correspondre à différentes bases de données, ou à des schémas différents au sein d'une même base de données.

  • Fermer : Ce bouton permet de refermer la barre d'outils.
Dans la partie supérieure du Centre de commande, la zone de texte Filtrer la liste permet de saisir le nom ou une partie du nom d'un processus, d’un dossier ou d’une vue pour filtrer automatiquement le contenu de la liste active.
