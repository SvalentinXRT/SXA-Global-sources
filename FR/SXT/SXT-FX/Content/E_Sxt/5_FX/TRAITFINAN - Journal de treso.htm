Journal de trésorerie

Pour chaque transaction, vous avez la possibilité de consulter le détail des flux de trésorerie correspondants générés dans le module ou une synthèse dans le module . Cette fonctionnalité est disponible dans les pages de gestion des transactions de change au comptant, à terme et de swap.

Vous pouvez également consulter le journal détaillé dans le module .

Consultation rapide depuis la page de gestion des transactions

Vous pouvez afficher les informations principales des flux de trésorerie liés aux transactions de change au comptant, à terme et de swap.

Vous pouvez également modifier certaines caractéristiques des flux affichés.

Afficher le journal de trésorerie dans FX
  1. Affichez la page de gestion des transactions de change au comptant, à terme ou de swap en sélectionnant l'option correspondante dans le menu de l'onglet .

  1. Dans la page qui s'affiche, sélectionnez une transaction.
  2. En bas de la page, cliquez sur le lien Afficher le journal de trésorerie.

Les flux de trésorerie associés à la transaction sélectionnée s'affichent dans un tableau en bas de la page.

Modifier des caractéristiques du flux

Vous avez la possibilité de modifier les informations suivantes du flux : statut du flux, nature budgétaire, libellé et référence.

  1. Dans la page de gestion des transactions de change au comptant, à terme ou de swap, affichez le flux de trésorerie comme décrit dans la rubrique précédente.
  2. Cliquez sur le lien Modifier les données en bas à gauche du journal de trésorerie.
  3. Cliquez sur un champ pour le modifier.

Seuls les champs décrits ci-après sont modifiables.

 Masquer le journal de trésorerie

Lorsque le journal de trésorerie est affiché, cliquez sur le lien Masquer le journal de trésorerie en bas à droite de la page pour masquer le journal.

 Consultation du détail dans le module

Vous pouvez consulter l'édition du journal de trésorerie dans le module .

Consulter l'édition du journal de trésorerie
  1. Dans le menu accordéon du volet , sélectionnez le module pour afficher le menu du module .
  2. Dans le menu du module , sélectionnez l'édition Journal de trésorerie.

Si un filtre a déjà été paramétré pour cette édition, le journal de trésorerie s'affiche dans le volet principal.

Si aucun filtre n'a été configuré pour cette édition, c'est la page de création de filtre qui s'affiche. Consultez la documentation du module pour plus d'informations sur la création de filtre pour l'édition du journal de trésorerie.

Afficher la description de l'édition

L'en-tête affiche le type et la date de l'édition, le nom du filtre ainsi que les critères de filtrage définis dans le filtre actif :

  • période de la date de valeur, d'opération ou de mise à jour des flux,
  • entités,
  • unité d'affichage des montants,
  • affichage des décimales,
  • statut des flux (initial, prévisionnel, réalisé),
  • affichage du détail des montants ou des totaux.

Afficher uniquement les flux générés depuis les transactions de change

Pour afficher uniquement les flux générés depuis le module dans le journal de trésorerie du module , vous devez modifiez le filtre du journal : dans l'onglet Options de la page de modification du filtre, sélectionnez l'option dans la zone Origine.

Consultez la documentation du module pour des informations sur les autres options de filtrage pour l'édition du journal de trésorerie.
Vous pouvez également consulter et modifier les flux de trésorerie dans le module , en sélectionnant l'option Journal du menu dans le module . Consultez la documentation du module pour plus d'informations.
