SXT

Le module permet de gérer quatre types de transactions :

  • Les virements internationaux
  • Les virements commerciaux
  • Les virements domestiques
  • Les virements de trésorerie
  • Les virements de trésorerie internationaux

La gestion de vos transactions dans le module est globale : elle débute par la saisie des transactions - ou l'intégration des demandes ou des ordres de paiement existants - et s'achève avec la génération de fichiers de remises à destination des banques ou autres intermédiaires financiers.

Le module peut être ensuite relayé par le progiciel pour que les ordres de virement soient directement transmis à leurs destinataires. Si l'une de vos banques utilise la norme bancaire EBICS, vos fichiers pourront être préalablement cryptés par le progiciel de signature de Sage, .

Ce manuel traite d'un type de règlement particulier, géré dans le module  : le virement de trésorerie.
