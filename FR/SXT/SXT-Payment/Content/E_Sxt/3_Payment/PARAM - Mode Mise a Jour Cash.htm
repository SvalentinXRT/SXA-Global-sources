Mode de mise à jour de la Trésorerie

Le mode de mise à jour de la Trésorerie permet de comptabiliser les virements dans les prévisions de trésorerie. La mise à jour sera déclenchée lors de la génération de la remise en banque.

Configurer le mode de mise à jour de la Trésorerie
  1. Pour accéder à cette fonctionnalité, dans le volet , sélectionnez l'option Eléments de paiement dans le dossier du module .

La page de configuration des éléments de paiement s'affiche. Cliquez sur le lien Mode de mise à jour Cash.

La page Mode de mise à jour Cash s'affiche. Vous pouvez consulter les modes de mise à jour déjà créés s'il en existe, ou alors créer un mode de mise à jour.

  1. Sélectionnez un type de transaction dans la liste déroulante.
  2. Sélectionnez un code transaction accessible par le bouton de ce champ.
  3. Cliquez dans le bouton radio Par ordre ou Par remise pour définir le mode de mise à jour. Si vous choisissez l'option Par ordre, l'application comptabilisera une écriture de prévision par virement, avec le montant de chaque virement. Si vous choisissez l'option Par remise, l'application comptabilisera une écriture de prévision par remise de virements, avec un montant global.
  4. Sélectionnez une référence de flux et un libellé de flux dans les listes déroulantes des champs correspondants.
  5. Cliquez sur cliquez sur Appliquer pour valider votre opération et poursuivre vos créations ou cliquez sur OK pour enregistrer le nouveau mode de mise à jour et revenir à la liste des identifications créées.
Vous devez créer autant de mode de mise à jour que vous avez de code transaction.
Modifier le mode de mise à jour de la Trésorerie
  1. Sélectionnez l'option Eléments de paiement dans le dossier du module . Cliquez sur le lien Mode de mise à jour Cash.
  2. Sélectionnez l'élément à modifier dans la liste.
  3. Cliquez sur Modifier. La page de modification du mode de mise à jour sélectionné s'affiche.
  4. Modifiez le contenu des champs référence flux et Libellé flux, puis cliquez sur OK pour sauvegarder votre opération.
 Supprimer le mode de mise à jour de la Trésorerie
  1. Sélectionnez l'option Eléments de paiement dans le dossier du module . Cliquez sur le lien Mode de mise à jour Cash.
  2. Sélectionnez l'élément à supprimer dans la liste.
  3. Cliquez sur Supprimer pour supprimer le mode de mise à jour sélectionné.
  4. Confirmez votre opération en cliquant sur Oui dans la boîte de message qui s'affiche.
