Entité

Une entité représente une unité de suivi budgétaire. Selon votre organisation, une entité peut correspondre à une société, un département ou encore une activité particulière de votre entreprise.

Le paramétrage des entités comporte une étape indispensable : le paramétrage du calcul automatisé de la TVA.

L'entité est définie dans le référentiel de données commun de . Son paramétrage est accessible via le module .

Accéder à la page de paramétrage des entités
  1. Dans le volet , sélectionnez le module dans le menu accordéon.
    Les tâches du module s'affichent dans ce volet.
  2. Sélectionnez l'option Sociétés dans le dossier de ce module.
    La page de paramétrage des entités s'affiche avec la liste des entités existantes sous forme de tableau.

Le code de l'entité est affiché dans la deuxième colonne du tableau.

 Créer une entité
  1. Dans la page de paramétrage des entités, cliquez sur le bouton Créer.
    L'onglet Identification s'affiche dans la page de création. Cet onglet permet d'enregistrer les informations standard utilisées pour l'identification unique de l'entité.

  1. Dans cet onglet, saisissez le code et le libellé de l'entité.

Afficher la description des champs à renseigner

Code Saisissez un code unique identifiant l'entité.
Ce code ne pourra pas être modifié.
Code court Ce code est automatiquement alimenté à partir du code standard défini. Vous pouvez le modifier au besoin lors de la création.
Ce code doit être unique et ne pourra pas être modifié ultérieurement.
Libellé Saisissez une description pour l'entité
Complément Vous pouvez, si vous le souhaitez, saisir un complément de description pour l'entité

Cet onglet vous permet de saisir les paramètres de calcul automatique de la TVA.

  1. Dans la partie , renseignez les zones Calcul de la TVA, Sens et Priorité.

Afficher la description des champs à renseigner

Champ Description
Zone Calcul de la TVA
Flag Calcul de la TVA Activez cette case à cocher pour activer le calcul automatique de la TVA dans le budget de trésorerie.
Par défaut, la TVA est calculée avec un glissement d'un mois. Dans la feuille budgétaire de trésorerie, la TVA calculée est alors affichée dans deux lignes spécifiques : la TVA encaissée pour les recettes (code système VATCOLL) et la TVA à récupérer pour les dépenses (code système VATRETR).
Si cette option est désactivée, aucune TVA ne sera calculée dans le budget de trésorerie.
Prorata Renseignez ici le pourcentage de TVA récupérable pour une TVA décaissée en cas de dépense.
Ce champ est renseigné par défaut à 100%.
Zone Priorité
Société Activez cette case d'option pour que le paramétrage du champ Direction effectué au niveau de l'entité prévale sur celui effectué au niveau des natures budgétaires.
Code budgétaire Activez cette case d'option pour que le paramétrage du champ Direction effectué au niveau de la nature budgétaire prévale sur celui effectué au niveau entité.
Zone Sens
Encaissement Activez cette case d'option pour que le montant et la date de transaction inscrits dans le budget de trésorerie servent de base au positionnement de la TVA.
Cette option est généralement utilisée par les sociétés de services.
Débit Activez cette case d'option pour que le montant et la date de transaction inscrits dans le budget opérationnel servent de base au positionnement de la TVA.
L'option Débit est activée par défaut.
