Groupe d'entités

Le paramétrage d'un groupe d'entités vous permet ultérieurement (lors de la saisie de votre budget) d'afficher une vue consolidée des budgets d'entités appartenant au groupe créé. La vue consolidée consiste en une sommation des budgets opérationnels et des budgets de trésorerie d'entités d'un même groupe.

Le groupe d'entités est un élément du référentiel de données commun de . Son paramétrage est accessible via le module .

Vous pouvez paramétrer un groupe d'entités uniquement lorsque les entités le composant sont paramétrées.
Accéder à la page de paramétrage des groupes d'entités
  1. Dans le volet , sélectionnez le module dans le menu accordéon.
    Les tâches du module s'affichent dans ce volet.
  2. Sélectionnez l'option Groupes sociétés dans le dossier .
    La page de paramétrage des groupes d'entités s'affiche avec la liste des groupes d'entités existants sous forme de tableau.

Le tableau affiche le code et le libellé du groupe d'entités dans les deux premières colonnes.

 Créer un groupe d'entités

Pour créer un groupe d'entités, vous devez avoir préalablement paramétré deux entités. 

Pour plus d'informations sur la création d'entités, consultez la rubrique Entité.

  1. Dans la page de paramétrage des groupes d'entités, cliquez sur le bouton Créer.
    L'onglet Généralités s'affiche dans la page de création. Cet onglet permet d'enregistrer les informations standard utilisées pour identifier un groupe d'entités de façon unique.

  1. Saisissez un code et un libellé pour le groupe d'entités dans les champs Code et Libellé 1.
  1. Dans le champ Entité leader, sélectionnez une entité principale, comme suit :
  1. Cliquez sur le bouton pour afficher la liste des entités configurées.
  2. Dans la fenêtre contextuelle qui s'affiche, double-cliquez sur une ligne pour la sélectionner.
    Le code et le nom de l'entité sélectionnée s'affichent dans l'onglet Généralités.
Le choix d'une entité principale n'a pas d'impact sur la consolidation.
  1. Cliquez sur l'onglet Répartition. Cet onglet permet d'associer des entités au groupe.

  1. Cliquez sur le bouton Contenu pour sélectionner des entités.
    Une fenêtre contextuelle affiche la liste des entités existantes.

  1. Cliquez sur une ligne pour la sélectionner ou annuler sa sélection.
    Une marque de coche à gauche d'une ligne indique qu'elle est sélectionnée.
  2. Cliquez sur le bouton pour valider la sélection d'entités.
    La sélection s'affiche dans l'onglet Généralités.
  3. Pour chaque entité sélectionnée, saisissez un pourcentage de pondération dans la colonne Pondération.
  4. Le pourcentage total de pondération des entités doit valoir 100%.
La pondération n'a pas d'impact sur la consolidation.
  1. Cliquez sur le bouton :
  • Appliquer pour enregistrer le paramétrage du groupe et créer un autre groupe d'entités,
  • OK pour enregistrer le paramétrage du groupe et revenir à la liste des groupes d'entités paramétrés,
  • Annuler pour annuler le paramétrage du groupe.
