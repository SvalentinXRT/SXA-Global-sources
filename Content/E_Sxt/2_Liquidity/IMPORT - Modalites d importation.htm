Configuration des modalités d'importation de transactions

La configuration des modalités d'importation de transactions consiste à spécifier le seuil de totalisation des montants, c'est-à-dire la limite à partir de laquelle les transactions ayant la même nature budgétaire, la même date et la même devise sont importées individuellement et non fusionnées dans une transaction globale.

Vous pouvez définir des modalités génériques au niveau de la structure du fichier. Vous pouvez également créer des modalités spécifiques pour la même structure, et pour une entité et/ou une nature budgétaire spécifiques.

Accéder à la page de paramétrage des modalités d'importation

Dans le volet , sélectionnez l'option Conditions d'importation de budget dans le dossier du module .

La page de paramétrage des modalités d'importation s'affiche avec la liste des modalités paramétrées sous forme de tableau. Le tableau présente notamment le code de la structure, la feuille budgétaire cible et le mode d'importation (ajout ou remplacement).

Le tableau affiche également les modalités d'importation spécifiées : le code de la nature budgétaire, le code de l'entité, le montant et la devise du seuil de totalisation, ainsi que la date, la devise et le libellé de la règle de totalisation.

 Créer des modalités d'importation
  1. Dans la page de paramétrage des modalités d'importation, cliquez sur le bouton Créer.
    La page de paramétrage des modalités d'importation s'affiche.

  1. Saisissez les informations générales de la structure.

Afficher la description des champs

Champ Description
Zone Identification
Structure Ce champ affiche la structure de fichier associée aux modalités d'importation en cours de création.
Cliquez sur le bouton pour afficher la liste des structures de fichiers. Dans la fenêtre contextuelle qui s'affiche, double-cliquez sur une structure pour la sélectionner.
Le code et la description de la structure s'affichent dans le champ Structure.
Code budgétaire Ce champ est facultatif. Il vous permet d'associer les modalités à une nature budgétaire spécifique.
Cliquez sur le bouton pour afficher la liste des natures budgétaires. Dans la fenêtre contextuelle qui s'affiche, double-cliquez sur une nature budgétaire pour la sélectionner.
Le code et la description de la nature budgétaire s'affichent dans le champ Code budgétaire.
Entité Ce champ est facultatif. Il vous permet d'associer les modalités à une entité spécifique.
Cliquez sur le bouton pour afficher la liste des entités. Dans la fenêtre contextuelle qui s'affiche, double-cliquez sur une entité pour la sélectionner.
Le code et la description de l'entité s'affichent dans le champ Entité.
Zone Règles de totalisation
Les transactions ayant même nature budgétaire, même devise et même date peuvent être fusionnées dans une transaction globale.
Vous pouvez définir un seuil de totalisation qui établit une limite de montant à partir de laquelle les transactions ne sont plus fusionnées, mais importées individuellement.
Seuil de totalisation Ce champ est facultatif.
Pour sélectionner la devise du seuil de totalisation, cliquez sur le bouton pour afficher la liste des devises. Dans la fenêtre contextuelle qui s'affiche, double-cliquez sur une devise pour la sélectionner.
Saisissez ensuite le montant du seuil de totalisation dans le champ de saisie du montant.
Si vous activez les trois cases à cocher décrites ci-après, le seuil sera appliqué aux trois critères sélectionnés.
Date identique Activez cette case à cocher pour que la totalisation soit appliquée aux transactions effectuées à la même date.
Devise TRX identique Activez cette case à cocher pour que la totalisation soit appliquée aux transactions effectuées dans la même devise.
Libellé identique Activez cette case à cocher pour que la totalisation soit appliquée aux transactions ayant le même libellé.
  1. Cliquez sur l'onglet Codes budgétaires par défaut.

Cette page vous permet de paramétrer deux natures budgétaires par défaut pour les encaissements et les décaissements.

Le champ Dépense affiche le code et la description de la nature budgétaire affectée par défaut aux transactions dont le montant est négatif et le champ Recette affiche le code et la transaction de la nature budgétaire affectée par défaut aux transactions dont le montant est positif.

  1. Pour affecter une nature budgétaire aux encaissements ou aux décaissements, cliquez sur le bouton pour afficher la liste des natures budgétaires. Dans la fenêtre contextuelle qui s'affiche, double-cliquez sur une nature budgétaire pour la sélectionner.
    Le code et la description de la nature budgétaire s'affichent dans le champ Code budgétaire.

Le bouton Appliquer vous permet d'enregistrer vos modifications sans terminer l'étape en cours.

Lorsque vous avez terminé le paramétrage des modalités d'importation, cliquez sur le bouton OK pour valider votre saisie ou sur le bouton Annuler pour l'annuler.

La page de paramétrage des modalités d'importation s'affiche avec les nouvelles modalités d'importation en bas du tableau.
