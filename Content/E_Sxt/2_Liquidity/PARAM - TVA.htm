Taux de TVA

Le paramétrage du taux de TVA permet de calculer automatiquement les éléments de TVA dans le budget de trésorerie.

Vous définissez les taux de TVA à appliquer à chaque nature budgétaire.

Vous pouvez par exemple créer deux taux de TVA, l'un à 5,5%, l'autre à 19,6%, pour les appliquer en fonction de la réglementation en vigueur sur le type de produit acheté ou vendu.

Accéder à la page de paramétrage des taux de TVA
  1. Dans le volet , sélectionnez le module dans le menu accordéon.
    Les tâches du module s'affichent dans ce volet.
  2. Sélectionnez l'option Paramétrage TVA dans le dossier de ce module.
    La page Calcul TVA s'affiche avec la liste des taux de TVA paramétrés sous forme de tableau. Le tableau présente le code et le libellé de chaque taux de TVA.

 Créer un taux de TVA
  1. Cliquez sur le bouton Créer.

La page de création s'affiche.

  1. Dans la zone Code TVA, saisissez un code unique et un libellé pour le taux de TVA.
  2. Dans la zone Taux TVA, cliquez sur le bouton Ajouter taux.

Une ligne est automatiquement ajoutée à la liste des valeurs de taux, avec la date du jour comme date de début de valeur.

  1. Modifiez la date de début d'application de la valeur si nécessaire dans la colonne Date cours.
  2. Saisissez une valeur de taux comprise entre 0 et 100 dans la colonne Valeur cours.

Vous pouvez définir, pour un même taux de TVA, des valeurs différentes pour différentes périodes : par exemple, un taux de TVA à 19,6% à partir du 01/01/2010 et à 19,7% à partir du 31/12/2012.

  1. Pour ajouter une autre valeur de taux, cliquez de nouveau sur le bouton Ajouter taux et répétez les étapes précédentes.

Pour supprimer une valeur de taux, cliquez sur le bouton Supprimer.

  1. Cliquez sur le bouton :
  • Appliquer pour enregistrer le paramétrage du taux de TVA et créer un autre taux,
  • OK pour enregistrer le paramétrage du taux de TVA et revenir à la liste des taux de TVA paramétrés,
  • Annuler pour annuler le paramétrage du taux de TVA.
