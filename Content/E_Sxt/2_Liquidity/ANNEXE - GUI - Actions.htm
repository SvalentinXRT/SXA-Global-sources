Boutons d'action sur une liste de paramètres

Les boutons de la barre d'actions diffèrent selon le type de fonction à laquelle vous accédez.

Dans les pages de paramétrage affichant un tableau avec la liste des paramètres, la barre d'outils Actions se présente comme suit :

Cette barre d'outils est ancrée au-dessus du tableau de paramètres.

Consultez les rubriques ci-après pour plus d'informations sur les actions possibles à partir des boutons de cette barre d'outils.

Créer une ligne

Cliquez sur le bouton Créer.

La page de création de l'élément s'affiche, dans laquelle vous pouvez saisir les informations à enregistrer.

Les codes identifiants ne peuvent pas être modifiés après leur création.

 Modifier une ligne Supprimer une ligne Actualiser l'affichage

Cliquez sur le bouton Actualiser pour actualiser l'affichage.

 Modifier les données directement dans le tableau

Le mode modification directe vous permet de modifier directement les valeurs dans le tableau affiché.

  1. Cliquez sur le bouton Modifier dans la grille, puis, dans le menu déroulant qui s'affiche, cliquez sur l'option Lancer l'édition en mode grille pour activer le mode de modification directe.
  2. Saisissez vos modifications directement dans les cellules du tableau.
  3. Pour annuler vos modifications, cliquez sur le bouton Modifier dans la grille, puis, dans le menu déroulant qui s'affiche, cliquez sur l'option Annuler les modifications sur la grille.
  4. Pour enregistrer vos modifications, cliquez sur le bouton Modifier dans la grille, puis, dans le menu déroulant qui s'affiche, cliquez sur l'option Enregistrer les modifications sur la grille.
 Exporter les données vers Microsoft Excel Imprimer la vue
  1. Pour imprimer le tableau affiché, cliquez sur le bouton Imprimer.

Une version imprimable s'affiche.

  1. Avant d'imprimer cette version, parcourez les pages à imprimer pour vérifier le contenu en utilisant la barre de navigation et de zoom.
  2. Dans la barre d'actions, cliquez sur le bouton Configuration impression pour choisir une imprimante et spécifier les options d'impression.

La boîte de dialogue de configuration de l'impression s'affiche. Pour plus d'informations sur cette boîte de dialogue, consultez la documentation de votre système d'exploitation.

  1. Spécifiez les options d'impression, puis cliquez sur OK.
  2. Dans la barre d'actions, cliquez sur le bouton Imprimer.

L'impression de la page est lancée.

Vous pouvez également envoyer le tableau par courrier électronique en cliquant sur le bouton Notifier dans la barre d'actions.
  1. Cliquez sur le bouton Fermer pour réafficher la page de paramétrage.
