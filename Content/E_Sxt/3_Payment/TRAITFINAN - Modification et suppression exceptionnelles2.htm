Suppression de remises

Lors du traitement de vos remises, il est possible que malgré les diverses étapes de contrôle et de validation, une erreur soit commise sur une remise déjà constituée.

L'application vous offre la possibilité de supprimer les remises de votre choix au statut Constituée, par le biais d'une manipulation spécifique.

Supprimer une remise avec le statut Constituée
  1. Notez le numéro de la remise que vous souhaitez supprimer.
  2. Accédez dans le volet au module Eléments communs.
  3. Dans le menu Administration générale, sélectionnez le sous-menu Paiement et recette.
  4. Cliquez sur Procédure exceptionnelle dans le menu des tâches liées.

  1. Dans le champ Opération, sélectionnez le bouton radio remise.
  2. Renseignez le numéro de remise dans le champ Numéro, puis cliquez sur OK.

  1. Cliquez à nouveau sur OK dans la boîte de message qui s'affiche et revenir à la page d'accueil des Eléments communs.
En retournant dans le module , pour consulter la remise ayant fait l'objet de la procédure exceptionnelle, vous pourrez constater qu'elle n'est plus affichée dans la liste des remises : elle a été supprimée. Ainsi, vous avez la possibilité de créer une nouvelle remise avec les ordres appropriés (et éventuellement modifiés) avant de la remettre en banque. Pour plus d'informations sur la modification exceptionnelle des virements, consultez la rubrique Modification et suppression exceptionnelles de virements.

Une fois votre remise générée ou archivée, vous avez également la possibilité de la supprimer.

Supprimer une remise avec le statut Générée/Archivée

La suppression d'une remise avec ces statuts s'effectue en deux étapes.

Dans un premier temps :

  1. Notez le numéro de la remise que vous souhaitez supprimer.
  2. Accédez dans le volet au module Eléments communs.
  3. Dans le menu Administration générale, sélectionnez le sous-menu Paiement et recette.
  4. Cliquez sur Procédure exceptionnelle dans le menu des tâches liées.

  1. Dans le champ Opération, sélectionnez le bouton radio remise.
  1. Renseignez le numéro de remise dans le champ Numéro, puis cliquez sur OK.

  1. Cliquez à nouveau sur OK dans la boîte de message qui s'affiche et revenir à la page d'accueil des Eléments communs.
En retournant dans le module , pour consulter la remise ayant fait l'objet de la procédure exceptionnelle, vous pourrez constater qu'elle est affichée dans la liste des remises avec le statut Constituée.

Vous pouvez désormais procéder à la suppression de la remise, qui constitue la deuxième partie du processus de suppression.

  1. Sélectionnez la remise que vous venez de refaire passer au statut Constituée, dans le menu remise lié à un des types de virements gérés dans l'application.
  1. Cliquez sur le bouton dans la barre d'outils.
  2. Cliquez sur Oui dans la boîte de dialogue qui s'affiche pour confirmer la suppression.

  1. Cliquez sur OK pour valider le traitement et revenir à la liste des remises. Vous pouvez constater que la remise a été effacée de la liste.

Au delà de la suppression des remises où seule disparaîtra la remise constituée, mais pas les virements qui la composent, vous avez la possibilité de purger les transactions de paiement pour faire disparaître à la fois la remise et les transactions liées.

Purger les transactions de paiement
  1. Accédez dans le volet au module Eléments communs.
  2. Dans le menu Administration générale, sélectionnez le sous-menu Paiement et recette.
  3. Cliquez sur Purger transactions de paiement dans le menu des tâches liées.

  1. Définissez les différents critères d'affichage.
  2. Dans l'onglet Opération, vous avez la possibilité de sélectionner les types de transaction que vous souhaitez supprimer.
  3. Cliquez sur OK.

  1. Sélectionnez les virements à purger, puis cliquez sur le bouton Purger.

  1. Cliquez sur Oui pour valider votre action.
