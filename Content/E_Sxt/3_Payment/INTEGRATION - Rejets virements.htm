Rejet des virements

Cette fonction vous permet de consulter l'historique des règlements rejetés lors de l'importation d'un fichier provenant d'un système externe.

Les règlements dits rejetés ne sont pas intégrés, mais enregistrés dans un historique avec un motif de rejet spécifiant les informations absentes ou incohérentes avec le référentiel de données de .

Différents types d'erreur peuvent générer un rejet : information obligatoire absente ou sans équivalent dans la base de données (cas des informations répertoriées dans le référentiel, comme les codes devise, les codes société...).

L'historique des rejets comporte l'intégralité des informations sur les règlements rejetés ainsi que les motifs de rejet générés. Vous pouvez ainsi consulter cet historique, corriger les erreurs, puis réimporter ces règlements rejetés.

Avant d'effectuer la correction manuelle des règlements rejetés, vérifiez que la structure de fichier est paramétrée correctement. Pour plus d'informations, consultez la section Structure des fichiers échangés, à partir de la page 1.rubrique Structure des fichiers échangés.

Accéder à la page de gestion des rejets

Dans le volet , sélectionnez l'option Rejets des types de règlement dans le dossier du module .

La page de gestion des rejets s'affiche avec la liste des règlements rejetés sous forme de tableau.

Ce tableau présente les motifs de rejet et les informations sur les règlements.

Tous les règlements sont sélectionnés par défaut. Si vous souhaitez annuler cette sélection et sélectionner une ligne spécifique, désactivez la première case à cocher du tableau, puis activez la case à cocher en regard de la ligne à sélectionner.
Consulter le détail des règlements rejetés

Dans la page de gestion des rejets, sélectionnez les lignes à consulter puis cliquez sur le bouton Visualiser.

Les informations détaillées sur le premier règlement sélectionné et sur les motifs de rejet s'affichent dans une nouvelle page.

  • Si vous avez sélectionné plusieurs règlements pour la consultation, cliquez sur le bouton Écriture suivante pour consulter le règlement suivant.
  • S'il existe d'autres messages d'erreur non affichés dans la page en cours pour le règlement sélectionné, utilisez les boutons Message d'erreur précédent et Message d'erreur suivant pour naviguer entre les pages détaillant les messages d'erreur.
  • Pour revenir à la page de gestion des rejets, cliquez sur le bouton Annuler.
 Corriger manuellement des règlements rejetés
  1. Dans la page de gestion des rejets, sélectionnez un ou plusieurs règlements et cliquez sur le bouton Modifier dans la barre d'actions en haut de la page.

Les informations détaillées sur le premier règlement sélectionné et sur les motifs de rejet s'affichent dans une nouvelle page.

Chaque message d'erreur présente un champ de saisie permettant de corriger la valeur incorrecte.

  1. Corrigez chaque information erronée en saisissant une nouvelle valeur dans le champ de saisie ou en utilisant le bouton pour sélectionner une valeur dans une liste.
  1. S'il existe d'autres messages d'erreur non affichés dans la page en cours pour le règlement sélectionné, utilisez les boutons Message d'erreur précédent et Message d'erreur suivant pour naviguer entre les pages détaillant les messages d'erreur.
  2. Si vous avez sélectionné plusieurs règlements à corriger, cliquez sur le bouton Écriture suivante pour consulter le règlement qui suit.
  3. Une fois la correction terminée, cliquez sur le bouton OK pour enregistrer vos modifications ou sur le bouton Annuler pour annuler.

La page de gestion des rejets est réaffichée.

Une fois les règlements corrigés, vous pouvez lancer une tâche d'importation sur les règlements rejetés.

 Purger les règlements

Dans la barre d'actions de la page de gestion des rejets, cliquez sur le bouton Purge, puis cliquez sur OK dans la boîte de dialogue de confirmation de la purge.

L'historique des règlements rejetés est purgé.

Cette opération est irréversible. Avant de purger l'historique des règlements, assurez-vous que ces informations ne seront pas nécessaires ultérieurement.
Relancer l'importation des règlements
  1. Dans la barre d'actions de la page de gestion des rejets, cliquez sur Traitement, puis sur Importer.

L'importation est lancée et une nouvelle page s'affiche, indiquant la progression de la tâche.

  1. Une fois la tâche terminée, cliquez sur le bouton Fermer en bas de la page.

La page de gestion des rejets est réaffichée. La liste des règlements en erreur est mise à jour depuis la dernière importation.

Les règlements corrigés et réimportés sans erreur dans le module sont effacés de la liste.
