Paramétrage de l'importation

La configuration de l'importation comporte deux étapes :

  • la configuration de la structure du fichier à importer,
  • le paramétrage des modalités d'importation.

 

Dupliquer une structure de fichier

Vous pouvez créer une structure en dupliquant les paramètres d'une structure existante.

  1. Dans la page de paramétrage des structures, cliquez sur la structure à dupliquer, puis cliquez sur le bouton Dupliquer dans la barre d'actions.
  2. La page de création de structure s'affiche, avec les paramètres de la structure dupliquée. Un numéro identifiant est créé automatiquement pour la structure par incrémentation.
  3. Dans la zone Libellé, saisissez le libellé de la nouvelle structure.

Vous pouvez modifier les paramètres de la structure dupliquée.

Pour plus d'informations sur ces champs, consultez la rubrique précédente.

 Modifier les correspondances de valeurs entre fichier et base de données

Vous pouvez enrichir ou modifier en permanence les correspondances de valeurs entre fichier et base de données.

  1. Dans la page de paramétrage des structures, cliquez sur la structure à modifier, puis cliquez sur le bouton Modifier les correspondances dans la barre d'actions.
  2. La page de définition des tables de correspondance s'affiche, avec les correspondances paramétrées pour la structure sélectionnée.

Pour plus d'informations sur les champs de cette page, consultez la rubrique décrivant la création de structure de fichier à importer.
