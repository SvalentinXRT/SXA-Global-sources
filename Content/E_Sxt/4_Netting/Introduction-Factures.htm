Saisie et modification des factures

Le module propose des fonctions permettant aux utilisateurs de saisir facilement leurs factures en toute sécurité : la saisie est autorisée uniquement avec les droits d'accès requis.

Les factures sont saisies pour un cycle défini au préalable, mais il est possible qu'une facture puisse être transférée vers un autre cycle pour être reportée à une date ultérieure.

Pendant la durée d’un cycle, le centre a la possibilité de contacter les participants pour qu’ils valident les factures saisies. A cette occasion, des modifications peuvent être aisément intégrées dans les factures nécessitant des changements dans les données.

En parallèle, des factures peuvent être annulées en cas d'erreurs (même validées) ou de retour de marchandises facturées, notamment : le module va alors créer des avoirs pour les factures intragroupe, intégrées par la suite dans le processus de compensation.

Le module offre également la possibilité de gérer des factures de tiers pour le compte de participants : le centre gère les règlements pour les tiers et intègre les traitements dans l’ensemble du processus de compensation.
