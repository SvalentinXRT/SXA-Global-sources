Assistance technique

Si vous cherchez une réponse à une question d'ordre technique ou pratique sur le module de , consultez la documentation mise à votre disposition.

Si votre recherche s'avère infructueuse, vous pouvez contacter à l'adresse suivante pour obtenir des informations sur les types d'assistance technique fournie :

Immeuble le Colisée II

Avant de faire appel à nos services, vérifiez que vous disposez d'un temps suffisant pour résoudre votre problème.

Placez-vous devant votre ordinateur et lancez à l'endroit où le problème s'est produit.

Les informations suivantes peuvent vous être demandées :

  • le code d'accès qui vous a été fourni lors de votre premier appel,
  • le numéro de licence de votre produit,
  • les caractéristiques de votre plate-forme (environnement, produit, configuration),
  • le type de matériel utilisé,
  • vos nom et adresse ainsi qu'un numéro de téléphone de contact,
  • une description brève du problème, l'étape au cours de laquelle ce problème s'est produit, la manière dont vous avez tenté de le résoudre et les conséquences de cette tentative.
