Intégration des données externes

Cette rubrique présente l'importation, dans le module , des opérations commerciales. Vous pouvez ainsi réutiliser les opérations commerciales créées dans un autre système et éviter les doubles saisies.

Vous devez au préalable :

  • exporter, dans un fichier plat et depuis le système externe, les opérations commerciales créées dans ce système,
  • configurer, dans , la structure des fichiers qui seront importés.

L'importation consiste à intégrer, à la base de données, un fichier plat exporté à partir du système externe.

Le processus d'importation peut générer des rejets : les opérations dites rejetées ne sont pas importées, mais conservées dans un historique où vous pouvez les corriger, puis relancer leur importation dans .

Une fois les opérations commerciales importées, vous pouvez :

  • modifier celles-ci via la fonctionnalité de gestion des opérations commerciales,
  • les associer aux transactions de change.

Vous pouvez également créer manuellement vos opérations commerciales. Pour plus d'informations sur la création et la modification d'opérations commerciales, consultez la rubrique Opérations commerciales.
