Identification des clients

L'identification client est propre à certaines transactions. Cette information optionnelle peut être requise par les agences qui utilisent le format AFB320. Votre numéro d'identification vous est remis par l'agence.

La commande Identification client vous permet de saisir une fois pour toutes le code dont l'agence a besoin lorsqu'elle récupère vos fichiers de remises.

Configurer l'identification client
  1. Pour accéder à cette fonctionnalité, dans le volet , sélectionnez l'option Eléments de paiement dans le dossier du module .

La page de configuration des éléments de paiement s'affiche. Cliquez sur le lien Identifications client.

La liste des identifications déjà paramétrées s'affiche. Vous avez la possibilité de supprimer ou modifier une identification déjà existante, ou d'en créer une nouvelle. Pour créer une nouvelle identification client, cliquez sur Créer.

  1. Saisissez le code de l'agence qui reçoit les remises au format AFB 320 dans le champ Donneur d'ordre ou sélectionnez-le dans la liste accessible par un clic sur le bouton du champ.
  2. Procédez de la même façon pour renseigner le code du donneur d'ordre.
  3. Saisissez l'identification client dans la limite de 16 caractères alphanumériques.
  4. Cliquez sur cliquez sur Appliquer pour valider votre opération et poursuivre vos créations ou cliquez sur OK pour enregistrer la nouvelle identification et revenir à la liste des identifications créées.
 Modifier une identification client
  1. Sélectionnez l'option Eléments de paiement dans le dossier du module . Cliquez sur le lien Identifications client.
  2. Sélectionnez l'élément à modifier dans la liste.
  3. Cliquez sur Modifier. La page de détail de l'identification client sélectionnée s'affiche.
  4. Modifiez le contenu du champ Identification client, puis cliquez sur OK pour sauvegarder votre opération.
 Supprimer une identification client
  1. Sélectionnez l'option Eléments de paiement dans le dossier du module . Cliquez sur le lien Identifications client.
  2. Sélectionnez l'élément à supprimer dans la liste.
  3. Cliquez sur Supprimer pour supprimer l'identification client sélectionnée.
  4. Confirmez votre opération en cliquant sur Oui dans la boîte de message qui s'affiche.
