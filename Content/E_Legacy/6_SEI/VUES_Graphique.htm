Graphique

Le Centre de commande permet de sélectionner un Graphique à visualiser. Le graphique s'affiche alors dans la fenêtre principale.

Les groupes sont positionnés en haut de l’écran et le groupe actif est surligné en vert.

Pour zoomer sur un niveau d'information dans un graphique, sélectionnez un point de données (par exemple, une barre) puis appuyez sur le bouton Forage avant au bas de la page.

Le filtre créé s'affiche alors sous le nom du tableau.

Le bouton Forage arrière est maintenant disponible, permettant de revenir au niveau d'information supérieur.

L'ordre des groupes peut être modifié en effectuant un drag and drop d'une position à l'autre.
Modifier le type de graphique
  1. Dans la fenêtre principale, appuyez le bouton [Type de graphique] situé dans le coin inférieur droit.

  1. Sélectionnez un nouveau type de graphique dans le menu contextuel.

La fenêtre principale est actualisée et affiche les informations sous une nouvelle présentation.
