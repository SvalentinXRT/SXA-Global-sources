Zoom

Utilisez la barre d'outils Zoom pour agrandir ou pour réduire l'affichage de l'édition à l'écran :

Cette barre d'outils indique la taille de l'affichage par rapport à la taille réelle de l'édition en pourcentage.

Utilisez le bouton :

pour réduire la taille d'affichage,

pour agrandir la taille d'affichage.

 Recherche textuelle

Utilisez la barre de recherche pour rechercher du texte ou des montants dans une édition :

Pour effectuer une recherche dans une édition :

  1. Entrez le texte à rechercher dans le champ Mot-clé.

Gardez à l'esprit que la recherche s'effectuera sur l'expression exacte.

  1. Si vous souhaitez rechercher du texte ou un nombre uniquement, cliquez sur le bouton Options, puis activez l'option Rechercher Texte ou l'option Rechercher Nombre.
  2. Cliquez sur le bouton Rechercher suivant.

La première occurrence trouvée dans l'édition est surlignée en bleu.

  1. Pour chercher une autre occurrence, cliquez de nouveau sur Rechercher suivant.
 Changement de langue de l'édition

Utilisez la barre de langues pour changer la langue d'affichage des libellés dans l'édition :

Le drapeau indique la langue d'affichage en cours.

Pour afficher l'édition dans une autre langue, cliquez sur le drapeau pour afficher la liste des langues disponibles. Dans le menu contextuel qui s'affiche, cliquez sur une ligne pour la sélectionner.

L'édition est réaffichée dans la langue sélectionnée.
