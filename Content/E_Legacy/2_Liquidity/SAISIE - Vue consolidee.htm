Consolidation au niveau groupe d'entités

La consolidation vous permet de consulter une vue globale, en lecture seule, du budget au niveau groupe ou au niveau unité.

Cette fonctionnalité est disponible en modes prévision et analyse.

Si vous gérez un exercice par devise, cette fonction s'avère particulièrement utile pour afficher une vue consolidée de ce budget en une seule devise. La devise de consolidation est la devise d'affichage définie dans le filtre.

Si vos prévisions budgétaires sont enregistrées dans différentes devises, vous devez avoir préalablement défini un cours prévisionnel pour les différentes devises.
Pour plus d'informations sur les cours prévisionnels des devises en cas de budgets multidevises, consultez la rubrique Devises et cours budget.
Configurer l'affichage consolidé au niveau groupe
  1. Dans l'arborescence des exercices budgétaires (située dans le volet gauche de la page de saisie de budget), cliquez avec le bouton droit dans l'arborescence des exercices et, dans le menu contextuel qui s'affiche, sélectionnez l'option Sélectionner pour consolidation pour activer la fonction de consolidation.
  2. Appuyez sur la touche Ctrl et, tout en maintenant la touche enfoncée, cliquez sur différentes versions de budget pour effectuer une sélection multiple.

Si vous gérez un exercice par devise, vous pouvez sélectionner une version par devise pour obtenir l'affichage consolidé de votre budget en une devise. La devise de consolidation est la devise d'affichage définie dans le filtre.

Toutes les versions sélectionnées doivent avoir les mêmes périodes d'exercice. Par exemple, vous ne pouvez pas obtenir d'affichage consolidé d'une version de budget de l'année 2010 et d'une autre version de budget de l'année 2011.
  1. Une fois la sélection effectuée, relâchez la touche Ctrl et cliquez avec le bouton droit de la souris, puis, dans le menu contextuel qui s'affiche, sélectionnez l'option Consolider les exercices.

La page de sélection de construction de feuille budgétaire s'affiche.

  1. Dans le champ Sélectionner Construction, sélectionnez une construction de feuille budgétaire en utilisant le bouton , puis cliquez sur OK pour valider votre sélection.

La feuille budgétaire est réaffichée, consolidant les budgets sélectionnés.

Les montants sont accessibles en lecture seule.

 Configurer l'affichage consolidé au niveau unité
  1. Dans l'arborescence des exercices budgétaires (située dans le volet gauche de la page de saisie de budget), cliquez avec le bouton droit de la souris dans l'arborescence des exercices et, dans le menu contextuel qui s'affiche, sélectionnez l'option Sélectionner pour consolidation pour activer la fonction de consolidation.
  2. Appuyez sur la touche Ctrl et, tout en maintenant la touche enfoncée, cliquez sur plusieurs unités pour effectuer une sélection multiple.

Si vous gérez un exercice par devise, vous pouvez sélectionner une unité par devise pour obtenir l'affichage consolidé de votre budget en une devise. La devise de consolidation est la devise d'affichage définie dans le filtre.

La période d'exercice doit être la même pour toutes les unités sélectionnées et les unités doivent appartenir à des entités différentes. Par exemple, vous ne pouvez pas obtenir d'affichage consolidé d'une unité pour l'année 2010 et d'une autre unité pour l'année 2011.
  1. Une fois la sélection effectuée, relâchez la touche Ctrl et cliquez avec le bouton droit de la souris, puis, dans le menu contextuel qui s'affiche, sélectionnez l'option Consolider les exercices.

La page de sélection de construction de feuille budgétaire s'affiche.

  1. Dans le champ Sélectionner Construction, sélectionnez une construction de feuille budgétaire en utilisant le bouton , puis cliquez sur OK pour valider votre sélection.

La feuille budgétaire est réaffichée, consolidant les budgets sélectionnés.

Les montants sont accessibles en lecture seule.

 Annuler l'affichage consolidé

Cliquez avec le bouton droit de la souris dans l'arborescence des exercices budgétaires (dans le volet gauche de la page de saisie de budget) et, dans le menu contextuel qui s'affiche, sélectionnez l'option Affichage normal de l'exercice.
