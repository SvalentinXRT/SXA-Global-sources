Volets

La page de saisie de budget affiche par défaut un volet à gauche de la page avec :

  • en haut, l'arborescence des exercices budgétaires, présentant les éléments sélectionnés dans le filtre d'affichage appliqué,
  • en bas, la configuration simplifiée de filtre.

Pour plus d'informations sur la configuration simplifiée de filtre, consultez la rubrique Filtres d'affichage.

Afficher ou masquer le volet gauche

Cliquez sur le bouton Configurer dans la barre d'actions en haut de page et, dans le menu contextuel qui s'affiche, sélectionnez l'option Afficher / Masquer le panneau.

 Afficher ou masquer l'arborescence dans le volet gauche

Cliquez sur le bouton Configurer dans la barre d'actions en haut de page et, dans le menu contextuel qui s'affiche, sélectionnez l'option Afficher / Masquer l'arborescence des révisions.

 Afficher ou masquer le filtre simplifié dans le volet gauche

Cliquez sur le bouton Configurer dans la barre d'actions en haut de page et, dans le menu contextuel qui s'affiche, sélectionnez l'option Afficher / Masquer le filtre résumé.
