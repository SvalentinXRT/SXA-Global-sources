Fonctionnement des filtres et des modes de visualisation

L'application offre la possibilité de séparer la sélection des données à afficher (via le filtre), de la manière dont les données sont affichées (via le mode de visualisation).

Filtre sur les données

Le filtre permet de définir les champs dont les données devront être affichées. Exemple : Sociétés, Banques...

Les opérateurs disponibles pour formuler les requêtes sont les suivants : Equal, Not equal, Contains, Like, In, Is Null et Is Not Null.

Exemple : Société EqualSage France et Banque Equal BNPP

Un seul filtre peut être défini par défaut à l'écran.
Modes de visualisation

Le mode de visualisation offre la possibilité de définir les colonnes, les tris, les regroupements, les détails, les agrégats, les groupements d’agrégats et les graphiques.

  • Colonnes
    • afficher ou cacher des colonnes via les actions + et -
    • définir le mode d'affichage en déplaçant les colonnes avec le curseur
  • Tri
    • par colonne
    • par ordre de tri
  • Regroupements
    • par ligne via les actions + et -
    • par ordre de regroupement en déplaçant les colonnes avec le curseur
Un seul mode de visualisation peut être défini à l'écran.
Filtre rapide

Les champs principaux candidats pour le filtre traditionnel sont également disponibles pour le filtre rapide.

Le filtre rapide peut-être sauvegardé en tant que filtre traditionnel.
