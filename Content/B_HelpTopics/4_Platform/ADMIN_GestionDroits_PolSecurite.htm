Politiques de sécurité

Dans le module , sélectionnez le menu Gestion des droits, puis cliquez sur Politique de sécurité.

La liste des politiques de sécurité pré-définies de s'affiche.

Ces politiques par défaut ne sont pas modifiables. Elles peuvent être directement affectées aux profils utilisateur ou être dupliquées pour créer des politiques personnalisées.

Dupliquer une Politique de sécurité
  1. Sélectionnez une ligne en cliquant sur la case à cocher correspondante, puis cliquez sur l'icône Afficher.

  1. Dans le menu Actions, sélectionnez Cloner.

  1. Dans l'onglet Informations Générales, saisissez un nouveau code, puis un libellé pour définir votre nouvelle politique de sécurité.
  2. Cliquez sur Enregistrer ou modifiez le paramétrage dans les onglets Sécurité et Autorisations d'accès (voir ci-dessous).

Vous pouvez également créer une politique de sécurité de A à Z en utilisant le bouton Créer de l'écran Administration / Gestion des Droits  / Politique de Sécurité.

L'écran de création se compose de trois onglets :

Informations générales
  1. Saisissez un code libre dans la rubrique Code pour nommer la nouvelle politique.
  2. Saisissez un intitulé dans la rubrique Libellé pour décrire cette politique.

  1. Cliquez sur le bouton Enregistrer.
 Sécurité
