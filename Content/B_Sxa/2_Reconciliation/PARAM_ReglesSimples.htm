Règles de rapprochement simples

Les règles simples permettent le rapprochement entre une écriture comptable et une écriture bancaire. Pour cela, vous devez établir des correspondances entre les codes d’opérations existant dans votre fichier comptable et les codes d’opération bancaires.

Exemple :

Une écriture comptable passée sur le code d’opération Chèques va se rapprocher avec une écriture bancaire passée sur le code bancaire 01 chèques payés.
  1. Dans le menu Rapprochement, sélectionnez Paramétrage puis Règle de Rapprochement.

L'écran Règles de Rapprochement s'affiche.

  1. Cliquez sur Créer Règle de Rapprochement.

Le formulaire de création de règle de rapprochement s'ouvre sur l'onglet Principal.

Cet onglet se compose de quatre rubriques :

Général
Les options Relevé Bancaire : Un et Relevé Comptable : Un sont activées par défaut. Les options Groupe et Exact vous permettent de réer des Règles avancées.
  1. Dans la zone Code, saisissez l'identifiant de la règle (format libre).
  2. Dans la zone Libellé, saisissez son intitulé.
  3. Dans la zone Relation Relevé, sélectionnez une des trois options disponibles :
Explicite

Ce type de règle de rapprochement classique, est basé sur la notion de lien Code Interbancaire CIB/Code Nature Comptable et Montant.

  1. Cliquez sur le bouton Sélectionner de gauche, correspondant à la rubrique Relevé Bancaire.

Une fenêtre de sélection s'affiche.

  1. Sélectionnez la ligne correspondant au Code Interbancaire à associer à votre Règle de Rapprochement, puis cliquez sur Accepter.
  2. Cliquez sur le bouton Sélectionner de droite, correspondant à la rubrique Relevé Comptable.

Une fenêtre de sélection s'affiche.

  1. Sélectionnez la ligne correspondant au Code Nature Comptable à associer à votre Règle de Rapprochement, puis cliquez sur Accepter.

Critères

Le paramétrage de critères de rapprochement supplémentaires permet d’optimiser le rapprochement automatique.

  1. Pour ajouter des critères de rapprochement, cliquez sur le bouton Ajouter, à gauche pour les critères bancaires et à droite pour les critères comptables.

Le nombre de critères de rapprochement supplémentaires dans une Règle de rapprochement est illimité.
  1. Saisissez l'intitulé du critère dans la zone Nom, puis cliquez sur le bouton de sélection dans la zone Propriété.

  1. Cliquez sur une ligne dans la fenêtre de sélection pour ajouter le critère.

L'option Fonction vous permet d'affiner le critère de sélection en recherchant une chaîne de caractères spécifique. En cliquant sur le bouton d'activation Non/Oui, une liste déroulante permet de choisir le type de balayage de caractères :

  • Sous-chaîne : vous devez renseigner la position de départ et la longueur de la chaîne
  • Troncature : vous devez sélectionner le symbole et la position de troncature (A gauche, A droite, Les deux).

 

 Dépendances de critères

Une fois les critères de rapprochement définis, cette rubrique permet d'ajouter au paramétrage des conditions de critère obligatoire ou optionnel.

  1. Cliquez dans la première case du tableau et sélectionnez un des critères bancaires définis dans la rubrique précédente.
  2. Cliquez dans la troisième case et sélectionnez un des critères comptables définis dans la rubrique précédente.
  3. Cliquez dans la deuxième case pour définir l'opérateur de comparaison entre les deux critères :
  • =
  • >
  • <
  • >=
  • <=

Par défaut, la relation de dépendance entre deux critères est obligatoire. Activez le bouton Facultatif : Oui pour la rendre optionnelle.
