Creating Templates

Before creating the first template, you need to create a transaction using the normal procedure. To open the transaction input dialog box, proceed as follows:

  1. click Payment Request (if you want to work on a payment request) in the Transactions menu, then click Create,
  2. click the category of transfer (if you want to work on a transfer) in the Transactions menu, then click Create.

A dialog box is displayed providing you with the list of already created templates.

For the very first creation, there is no predefined template. So, you need to select this option.
  • From a pre-existing template: This option must be selected. Click once to select the right template from the list.

  • New <Transaction Category>: This option allows creating a transaction from a blank form.
  1. After selecting one of these options, click OK to access the entry page.

  • The Save as Template... button enables you to keep your information in a reusable template.

If earlier you selected a template, the page is displayed with the corresponding information, otherwise the fields are blank. For more information on how to create transactions, see the Transfers Follow-up topic.

With or without a template, you can keep your information.

  • The Save as Template... button displays the corresponding page, in which you can enter a code and description for your new template.

  1. Click OK to save your new template. Do not forget to save the transaction too, otherwise it will not be taken into account.

You can replace a template by a new one later on. You need to enter a code or select it from the list – clicking in the Code field. Upon saving the new template, a message box will inform you that this template code already exists and will prompt you to confirm the update.

Creating Entry Templates

After opening and completing the creation/modifcation page for Transfers:

  1. click Save as Template...,
  2. in the Save as Template... page, you have two options:
  • enter the code of your new template (limited to 8 alphanumerical characters), and its description (limited to 35 alphanumerical characters),
  • or select the code of an existing template to replace it, using in the Code field, and double-click the template to overwrite,
  1. click OK to validate your entry.

If you replaced the new template by another one, you also need to confirm your choice, by clicking Yes in the Save Template... dialog box.

 Creating Transactions from Templates

From the page listing the transaction types that you are working on (payments or transfers), follow these guidelines:

  1. Select the transaction creation command – Treasury Transfer in this example.
  2. Complete the page if you want to restrict the number of already created elements, then click OK.
  3. In the first creation dialog box – Create Treasury Transfer in this example – click in the From a pre-existing template field.
  4. Select your template from the Templates dialog box, then click OK.

  1. Confirm the use of the template clicking OK in the first creation page.

The creation page for transfers is displayed. By default, it recaps all information included in the template. You need to complete the empty fields and to adapt certain elements of the template (Transaction Amount, for instance) to your new transaction.
