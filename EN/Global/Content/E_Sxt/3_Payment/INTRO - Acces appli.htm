Accessing

  1. Launch the application.

The identification dialog box of opens up.

  1. Select your workgroup, username and password, then click OK.

If some of these elements are missing or incorrect, contact your administrator.

The application window is displayed.

  1. In the accordion menu of the pane or the pane, select the module.

The list of the module functions appears above the menu.

If you used the pane, the homepage set up for the module is displayed. For more information on the setup of the homepage for the application modules, see the documentation.

To get to know the application, see the topicschapters that follow the logical order of this application:

  • Setup of the Working Environment,
  • Transaction Import,
  • Management of Payment Requests and Treasury Transfers,
  • Transaction Export,
  • Generation of Reports.
