 Text Search

The search bar enables you to look for text or amounts in the report:

To search the report:

  1. Enter the text to find in the Keyword field.

Remember that this function searches for the exact expression.

  1. If you want to search for text or numbers only, use the Options button, then enable the Find Text option or the Find Number option.
  2. Click Find Next.

The first hit found in the report is highlighted in blue.

  1. To search for another hit, click Find Next again.
 Changing the Report Language

The language bar enables you to change the display language for the report descriptions.

The flag shows the current language.

To view the report in another language, click the flag to display the list of the available languages. In the contextual window, click a line to select it.

The report is redisplayed in the selected language.
