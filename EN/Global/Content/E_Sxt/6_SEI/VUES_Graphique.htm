Charts

The Command Center enables you to view a Chart. When you select a chart, it is then displayed in the main window.

The various groups are displayed at the top of the screen, the active current group is highlighted in green.

To focus on a level of detail, select the data point (for eg. a bar) and use the Drill down function.

This new filter is displayed under the table name.

The Drill up function is now available, enabling you to go back to the last level of detail.

You can modify the order of the groups by dragging and dropping them wherever you want.
Modifying Chart Types
  1. In the main window, click the arrow of the button which specifies the chart type in the lower right corner.

  1. From this new contextual menu, select another chart type.

The main window is refreshed with the new layout.
