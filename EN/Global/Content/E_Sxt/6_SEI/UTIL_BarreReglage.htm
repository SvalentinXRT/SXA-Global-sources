Toolbar

The Toolbar pane grants access to the most used options.

To open it, click the button on the top right of the window.

  • Log out: This button logs you out of application.
  • Open External Link: You can directly access a view or integrate it into another application.
This extra option is part of the Mobile license.
  • Open External Link: You can directly access a view or integrate it into another application.
This extra option is part of the Web license.
  • Export: This menu enables view exports into PDF, Excel or HTML.
  • Language: This submenu enables you to modify the language for data display. English is the default language.
  • Environment: This option enables the selection of another data source.
  • Close: This button closes the Toolbar.

Depending on usage, some options are not available. These options may be disabled for the following reasons:

  • The current user has no access rights to this option.
  • The option cannot apply (for e.g.. if no view has been opened, the Export Data option is not available).
