Login Window

When is accessed via any web browser, the login window is displayed. By default, it connects you to the pre-defined Central Point on the server.

The top portion of the login window shows the actual Central Point where you are going to log in. You may select another Central Point from the drop-down list when clicking the button . It also allows you to modify the language.

  • Enter your domain name: It must be a valid domain.
This value is optional and should be used only if the user profile requires it.
  • Enter your username: It must be a valid user profile with authorized access to .
  • Password: It must be the valid password of a user.
  • Keep me signed in: Select this option if you want this connection to remain active for a period of two weeks.
  • Connection: When you have defined all the connection parameters, click this button to start the application and display main window.
