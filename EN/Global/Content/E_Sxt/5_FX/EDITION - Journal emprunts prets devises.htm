Currency Borrowings/Loans Ledger

This report enables you to view the details of the currency borrowings and loans posted in . The currency borrowings and loans are taken into account for the calculations of the Position Worksheet.

Accessing Currency Borrowings/Loans Ledger
  1. In the pane, select the Currency Borrowings/Loans Ledger option in the folder of the module.
  • If no filter has been set up for this report, the creation page for filters is displayed. You first need to create a filter.

This filter enables you to select the transactions of currency borrowing and loan to display, according to your criteria.

View the topic below for more information on the creation of selection filters for transactions.

  • If a filter has already been set up, the modification page for filters appears with the default filter settings.

If you need to modify the filter settings, see below the topic regarding filters creation.

  1. Click OK to view the report with the filter settings.

Creating Display Filter for Currency Borrowings/Loans Ledger
  1. To access the creation page for filters from the report page for the Currency Borrowings/Loans Ledger, click the Actions dropdown menu in the filters toolbar, and the Add option in the menu.

The creation page for filters is displayed.

This page enables the definition of unique identification information for the filter.

  1. Enter a code and a description for the filter in the Code and Description fields.
  2. If you want this filter to be applied by default when you access the report page, enable the Template Default Filter option.
  3. If you want this filter to be exclusively accessed by yourself, select the Private option.
  4. You may add a comment in the Text area.
  5. Click the Configure button to set up the filter settings.

The configuration page for filter settings is displayed.

If you do not specify any filtering criterion in the different tabs of this page, the report will include all the existing commercial transactions.

The Entities tab enables you to specify filtering criteria on the companies linked to borrowing and loan transactions to include in the report, as well a filtering criterion on the counterparty and the intermediary.

  1. In the Select area, enable the following option:
  • All not to apply any filter on the companies,
  • Single to select a single company (selection in the area below),
  • Multiple to select several companies (selection in the area on the right).
  1. In the Select area , if you selected the following option:
  • Single, select a company by clicking the button in the area below, then, in the contextual window, double-click your company.
  • Multiple, select several companies as described below.
  1. On the right of the Select area, click the Select button.
  2. In the contextual window, enable the options in front of the codes for the companies to select, and click the button at the top of the window.

The codes and descriptions for the selected companies are displayed in the table of the area.

  1. In the Additional Criteria area, you can add another filtering criterion on the counterparty and the intermediary, using the button as described below.

The Main Currencies and Format tabs in this page are the same tabs as those in the creation page for filters of the Exchange ledger, apart from the field described below. For more information on the specification of filtering criteria and report settings in these tabs, click here.

The Format tab includes an additional area, the Conversion area, which enables you to select a conversion currency for the countervalue amounts, using the button as explained earlier. If no conversion currency was selected, the totals and subtotals only are converted into the reference currency.

  1. Click the Period tab.

This tab enables you to specify filtering criteria on the transaction date, start and end dates of the borrowings and loans to include in the report.

  1. To define a filtering process on the transaction date:
  1. Enable the option TRX Date.

The fields next to the option are enabled.

  1. In these fields, specify the start and end date of the period, between which must fall the transaction date of the transactions to include in the report, by going through the following process:
  • In the entry field, select the value to modify and enter a date with the format dd.mm.yyyy.

- or -

  • Click the button to display the calendar.

A contextual window displays the calendar.

View Calendar

Use the arrows and to select the month, year and day.

- or -

To set the start date to today's date, click the button Today.

  1. To specify a filtering process on the delivery date, enable the Delivery Date option, then go through the same process as above, to define the start and end dates of the period, between which must fall the delivery date for the transactions to include in the report.
  2. Go through the same process if you want to define a filtering on the end date of the borrowing and loan transactions.

The Options tab of this page is the same one as the tab in the creation page for filters of the Exchange Ledger. For more information on the specification of filtering criteria and report settings in this tab, click here.

  1. Click the Transactions tab.

This tab enables you to specify filtering criteria on the type of the borrowing and loan transactions to include in the report.

  1. In the Select area, enable the following option:
  • All not to apply any filter on the transaction types,
  • Single to include only the borrowing and loan transactions with the selected type (in the area below),
  • Multiple to include the borrowing and loan transactions with the selected type (in the area on the right).
  1. Go through the same process as in the Entities tab, to select one or several transaction types.
  1. In the Direction dropdown list, select a filtering criterion for the direction of the transaction:
  • Investment to select loans only,
  • Financing to select borrowings only,
  • Investment & Financing to disable any filter on the transaction direction.

The Advanced tab of this page is the same one as the tab in the creation page for filters of the Exchange Ledger. For more information on the specification of filtering criteria in this tab, click here.

  1. Click the following buttons, as needed:
  • OK to save the selected setup,
  • Cancel to view the Currency Borrowings/Loans Ledger without saving this setup.

The creation page for filters is redisplayed.

  1. Click OK to confirm the filter creation and view the report with the new filter settings.
