Cash Ledger

For each transaction, you can view the details of the cash flows generated in or a recap in . This function is available from the management pages for Spot Exchange, Forward Exchange and Swap Transactions.

You may also view the detailed ledger in the module.

Quick View from Transaction Management Page

You can view the main information elements on the cash flows linked to Spot Exchange, Forward Exchange and Swap Transactions.

You may also change some characteristics of the displayed flows.

Viewing Cash Ledger from FX
  1. Go to the management page for Spot Exchange, Forward Exchange and Swap Transactions by clicking the corresponding option in the menu of the tab.

  1. Select a transaction in the new page.
  2. Click the link Show Cash Ledger at the bottom of the page.

,The cash flows associated with the selected transaction are displayed in a table, at the bottom of the page.

Modifying Flow Characteristics

You can change the following data of the flow: flow status, budget code, description and reference.

  1. In the management page for Spot Exchange, Forward Exchange and Swap Transactions, display the cash flow as described in the previous topic.
  2. Click the Edit Data link at the bottom left of the Cash Ledger.
  3. Click a field to modify it.

You can edit only the following columns.

Field Description
N

Click the field to modify the flow status, then click the button in this field. A contextual window containing the following values, opens up.

  • O: initial flow
  • F: forecast flow
  • A: actual flow

Click a line to select it.
Budget Code Click the field to modify the budget code, then click the button in this field. A contextual window shows the list of the available budget codes. Double-click a line to select it.
Description Enter a short description for the processed flow.
Reference Enter the external reference of the flow.

Once you completed the entry, click Save Changes to save your modifications, or Cancel Editing to cancel them.

 Hiding Cash Ledger

In the Cash Ledger, click the Hide Cash Ledger link at the bottom right to hide the ledger.

 Viewing Details in

You can view the report of the Cash Ledger in the module.

Viewing Cash Ledger Report
  1. In the accordion menu of the pane, select to display the menu.
  2. In the menu of , select the Cash Ledger report.

If a filter has already been set up for this report, the report page opens up, presenting the Cash Ledger in the main pane.

If no filter has been set up for this report, the creation page for Filters is displayed. See the documentation for more information on how to create Filters for the Cash Ledger Report.

View Report Description

The header presents the type and date of the report, the filter name, as well as the filtering criteria defined in the current filter:

  • periods of the value date, transaction date or update date for the flows,
  • entities,
  • display unit for the amounts,
  • display of decimals,
  • flows status (initial, forecast, actual),
  • view of the amounts' details or totals.

Displaying Flows created from FX Transactions only

To display only the flows created from the module in the Cash Ledger, you have to change the ledger filter: in the Options tab of the filter's modification page, select the option from the Origin field.

See the documentation for more information on other filter options for the Cash Ledger Report.
