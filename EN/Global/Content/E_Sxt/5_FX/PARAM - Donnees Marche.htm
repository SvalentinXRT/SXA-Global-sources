Module

You also need to set up the following exchange rates. For more information on the setup process, see the module documentation.

  • Budget Rate: monthly forecasts for the supported currencies' rates. There is only one budget rate per currency, for all the existing companies.
  • Market Rate: exchange rates used on the markets
  • Rates Curve: values of the rates associated with a currency for the calculation of its forward rates
  • Currencies/Rates Equivalences: association of reference rates with currencies used if no rates curve set up
