Manual Update

You can settle your commercial transaction manually in both Global, Decrementation Mode and Global, Budget Decrementation Mode, when automatic update was not possible or when the amount of a forecast is bigger than the actual flows (a non-null amount in position) and that no more bank actual is expected.

This settlement which is also called update, can be total or partial. The update is partial when the settlement is performed only on a part of the amount.

Once completed, the forecasts cannot be modified anymore. considers that all completed transactions are actual transactions.

To view the list of the updated or non-updated commercial transactions, you simply need to set up the display filter in the management page for Commercial Transactions.

Updating Commercial Transactions Manually

As the update is automatic in Detailed, Reconciliation Mode, this function is only available for the Global, Decrementation Mode and Global, Budget Decrementation Mode.

  1. In the management page for Commercial Transactions, enable the options corresponding to the transactions to update, then click the Settle button in the actions toolbar.
This function is only available for the transactions with the "Open" or "Updated" status.

For more information on how to access the management page for Commercial Transactions, see the Commercial Transactions topic.

The page for Manual Update of Commercial Transactions opens up.

For more information on the displayed fields, see the Commercial Transactions topic.

  1. If you want to modify the link number assigned to the commercial transaction, in the Link N° field, click the button , then, in the contextual window, double-click a line to select the corresponding link number.
  2. In the Date of Rate field in Settle, enter a date for the exchange rate or use the calendar. For more information on how to use the calendar, click here.

The default value for this field is today's date. The date for the Exchange Eate must be after the Transaction Date and before the Delivery Date.

The value of the Exchange Rate is then updated according to the date specified in this field.

  1. If you want to modify the exchange rate value, enter the new value in this field.
  2. In the Amount field, enter the settlement amount expressed in the main currency.
The default value for this amount is the transaction amount in position. It cannot be higher than the initial transaction amount.
  1. If you selected several transactions to update, click the Next button in the actions toolbar to enter the data which enable the update of the next transaction.
  2. Once you entered all the data required for the update of the selected transactions, click OK in the actions toolbar to validate the update or Cancel to cancel it.

The management page for Commercial Transactions appears.

Once you validated the settlement:

  • if the settlement amount equals the amount of the commercial transaction, the update is total and the transaction status is set to "Closed",
  • if the settlement amount is lower than the amount of the commercial transaction, the update is partial and the transaction status is set to "Updated."

After the update validation, the update amount is substracted from the amount in position. For a total update, the amount in position equals 0.

You can view the settlement process in the Payments View tab, that can be accessed by clicking Show Details at the bottom of the management page for Commercial Transactions. For more information on this function, see the procedure for Payments Display.
