Importing G/L Data

The module enables you to import G/L data from external systems to feed automatically your budget, which avoids unnecessary manual entries.

You can import data from a flat file. This file can be exported from an accounting system.

Data import consists in several steps:

  • Setting up import
  • Running data import
  • Correcting the rejected data and re-importing the corrected data

After importing data, you can still enter and calculate data in the budget sheets. See the Budget Sheet Entry and Analysis topic for more information on these functions.
