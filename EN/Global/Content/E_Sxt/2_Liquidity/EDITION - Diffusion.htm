Saving, Notifying, Printing

Saving

The module enables you to save the current report under different file formats, in specified locations.

  1. In the actions toolbar of the report page, click the Export Report option.
  2. In the dialog box, select the final format of the file and the location of the generated file, then click OK.

Reports can be saved under different formats: CSV, HTML, ODBC, PDF, REC, RPT, RTF, TTX, TXT, XLS, XML.

 Forwarding by E-mail

The module enables you to generate a HTML or PDF file and to send it by e-mail.

  1. In the actions toolbar of the report page, click the Notify option.
  2. In the dialog box, select the final format for the e-mail, then click OK.

A contextual window shows the progression of the task process.

Your messaging service is automatically started. The generated file under the selected format is attached to a new message.

 Printing
  1. In the actions toolbar of the report page, click the Print Setup option.

The setup dialog box for the printing opens up.

Enter:

  • the size and source of the paper in the printer,
  • the paper orientation (portrait or landscape),
  • the dimensions of the printing margins.
  1. Select a printer and click OK.
  2. In the actions toolbar of the report page, click the Print option.

The setup dialog box for your printer opens up and enables you to launch the printing.

For more information on this dialog box, see the documentation of your operating system.
