Currencies and Budget Rates

There are two different types of currency rates in :

  • the market rate which is the actual currency rate used by the module for flows,
  • the budget rate which is a forecast rate used by the module for budget transactions.

There may be a difference between the budget rate and the market rate. You can view this difference in the discrepancies displayed on the entry page for the budget (analysis mode) or in the budget sheet report.

The setup of currencies is required before creating budget sheets. When setting up a company (see the Organizational Structure topic), you need to select a currency.

The budget rate must be set up if you work with several currencies. For example, when a group consists of companies for which the budget is in different currencies, the budgets of the companies can be grouped in a consolidated view at the level of the group and in a unique currency.

The value for the budget rate is defined on a monthly basis.

The currency can be accessed via the module and the budget rate via the module.

Defining Currencies
  1. Accessing Setup Page for Currencies:
    1. In the pane, select the module from the accordion menu.
      The tasks of the module are displayed in this pane.
    2. Select the Currencies option in the folder of this module.

    The setup page for currencies is displayed with the list of currencies in a table.

    The table shows the code and description of each currency in the second and third columns.

  2. Creating Currencies:
  1. Click the Create button.
    The General tab of the creation page for currencies is displayed.

  1. Select a code in the Code dropdown list of the Identification area.
  1. Click the following buttons, as needed:
  • Apply to save the setup of the currency and create another one,
  • OK to save the setup of the currency and display again the list of set currencies,
  • Cancel to cancel the setup of the currency.

The next step, which is decribed below, consists in defining the budget rate between the new currency and the reference currency.

 Defining Budget Rates
  1. In the Modules Tasks pane, select the module from the accordion menu. The tasks of the module are displayed in this pane.
  2. Select the Budget Rate option in the file.

The setup page for budget rates is displayed with the list of the set up forecast rates in a table.

The codes and currency pairs are displayed, along with their forecast values for each month of the current year.

  1. To select a currency pair, enable the option on the left of a line and click the Change button.
    The detail form of the budget rate is displayed.

The Identification area displays the identification information of the currency for which you want to define the rate against the reference currency. It cannot be modified.

  1. In the Years area at the bottom right, select the year to set up.
    The budget rates for each month of the selected year are displayed in the Yearly Rate area.
  2. Enter the values of the budget rate for each month.
  1. If you want to copy the value of a month in the following months of the year, place the cursor in the cell of the value to copy, and click Copy.

The values of the following months are overwritten and updated with the copied value.

  1. Once you defined the budget rate for the selected currency pair, click OK to save your entry or Cancel to cancel the current setup.
