Filters for Exercises Comparison

  1. In the Filters toolbar on the exercises comparison page, click the Actions button, then select the Add option.

The creation page for filters is displayed.

This page enables the definition of unique identification information for the filter.

  1. Enter a code and a description for the filter in the Code and Description fields.
  2. Enable the Template Default Filter option, if you want this filter to be applied by default when displaying the budget sheet for exercise comparison.
  3. If you want this filter to be only accessible by yourself, select the Private option.
  4. You may add comments in the Text field (optional).
  5. Click the Configure button to set up the filter settings.

The configuration page for filter settings is displayed.

 

The Company tab enables the definition of the companies and the company groups for which the exercises are to be compared, as well as the budget sheet designer.

The exercises to compare could be selected later on in the Entities to compare tab.

The budget sheet designer enables the specification of the budget codes to display in the exercises comparison sheet.

  1. In the Select by area, select one of these options:
  • To display all the set companies, enable the All option.

All the exercises of the set up entities may be selected in the Entities to compare tab.

For more information on setting up budget Exercises, see the Budget Exercise topic.

  • Companies to filter the companies

The Select button is enabled in the Companies area. Click this button, then in the contextual window, enable the options corresponding to the entities that you want to select. Click the button to validate your selection.

The budget exercises of the selected entities may be selected in the Entities to compare tab.

  • To filter the company groups, enable the Company Groups option.

The Select button is enabled in the Companies area. In the contextual window, enable the options corresponding to the company groups that you want to select. Click the button to validate your selection.

The budget exercises of the selected entities may be selected in the Entities to compare tab.

  1. In the Designer area, click the button to display the list of the set up designers, then, in the contextual window, double-click a line to select the corresponding designer.

The codes and descriptions of the selected designer are displayed in the area.

The budget codes included in the selected budget sheet designer only are to be compared. To extend the budget code selection, you can either select another designer, or add budget codes into the structure associated with the designer. For more information on the setup of budget sheet structures, see the Budget Structures topic.

  1. Go to the Currencies tab.

This tab enables you to select the currencies to include. For example, if you are dealing with a budget in both euros and dollars, you can choose to view exercises in euros only.

  1. In the Select by area, select your option:
  • All to cancel filtering at this level
  • Multiple to enable filtering at this level

If you selected Multiple, the Currencies area is activated. In this area, click Select to display the set currencies, and in the new contextual window, select the options corresponding to the currencies to be included. Click the button to validate your selection.

The selected currencies are displayed in the Currencies area.

  1. Click the Period tab.

This tab enables you to select the exercise periods to be included. It also enables you to define the division into periods for the exercise layout.

The Budget periods area enables you to select the exercise periods to compare, and the Period STEP area enables you to define the division into periods for the exercise display.

  1. In the Period STEP, select the division into period that you want for the entry page display.

The default option is "Month", for a monthly division.

The exercises comparison page displays one column by period step, i.e. one column by month for a monthly division. Transactions are to be grouped according to the selected periodic division.

  1. In the Budget Periods area, click Select, then, from the new contextual window, select the options corresponding to the relevant exercise periods. Click the button to validate your selection.
The list only displays the periods for which at least one budget exercise is created.

The selected exercise periods are displayed in the Period Codes area.

  1. Click the Entities to compare tab.

This tab enables you to select budget exercises and their versions to compare (initial version, revision 1, etc.).

  1. In the Exercise selection 1 area, select the first group of budget exercises to compare as follows:
  1. Click Select, then, from the new contextual window, select the options corresponding to the relevant exercises and click OK to validate.
The contextual window only presents the exercises associated with the Entities and the periods selected in the Periods tab.

The codes and descriptions for the selected companies are displayed in the first table of the area.

The selected exercises must all be linked to the same exercise period.

The second table enables you to select the exercise version to be compared for each exercise.

  1. In each line (corresponding to a selected exercise) of the second table, select the version to be compared from the Status field.
  1. In the Exercise Selection 2 area, select the second group of budget exercises to be compared as for the first group.
  2. If you want to restrict the comparison to a shorter time span than the periods associated with the selected exercises, select a new period in the Dates area following this process:
  1. In the Start Date Exercise 1 field, enter the start date for first group of exercises or use the calendar. Click here for more information on how to use the calendar.
  2. In the End Date Exercise 1 field, specify the end date in the same way.

The start and end dates of the comparison period for the second group of exercises are automatically updated in the Start Date Exercise 2 and End Date Exercise 2 fields. The process is completely automatic.

  1. Click the Format tab.

This tab enables you to define the format information: display currency, display unit of the amounts and budget type.

  1. In the Conversion Currency area, select the conversion currency of the amounts.

Click the button to display the list of the set up currencies. In the contextual window, double-click the line to select it.

The codes and descriptions of the selected currency are displayed in the Conversion Currency area.

  1. In the Display BS Grids area, select your option:
  • OBS to display the comparison on the Operational Budget
  • TBS to display the comparison on the Treasury Budget
  1. From the Amount dropdown list, select the display unit for amounts. For example, you can choose thousands, if the amounts of your budget are significant.
  2. Click the Category tab.

This tab enables you to select the budget code categories to be compared.

  1. Select the check boxes in front of the budget code categories to be displayed in the exercises comparison.
  2. Click the following buttons, as needed:
  • OK to display the exercises comparison page applying the filter settings
  • Cancel to display the exercises comparison page without applying the filter settings

For more information on the application, modification and deletion of filters, see the Filters topic.
