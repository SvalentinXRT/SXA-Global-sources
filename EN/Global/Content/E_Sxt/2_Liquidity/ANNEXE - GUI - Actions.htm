Action Buttons over Settings Lists

The action buttons of the toolbar depend on the function that you access.

In the setup pages displaying a list of settings in a table, here is the kind of Actions toolbar that you will encounter:

This toolbar is docked above the settings table.

See the topics below for more information on the available actions from the buttons of this toolbar.

Creating Lines

Click the Create button.

The editable creation page of the element is displayed.

The identifiers cannot be modified later.

 Modifying Lines Deleting Lines Refreshing Page

Click theRefresh button to update the page.

 Modifying Data directly in the Table

The direct modification mode enables you to edit the values directly in the table.

  1. Click the Edit in Datasheet, then, in the dropdown menu, click the Start Grid Edit option to enable the direct modification mode.
  2. Enter your modifications directly in the cells of the table.
  3. To cancel these modifications, click the Edit in Datasheet button, then, in the dropdown menu, click the Cancel Grid Changes option.
  4. To save these modifications, click the Edit in Datasheet button, then, in the dropdown menu, click the Save Grid Changes option.
 Exporting Data to Microsoft Excel Printing View
  1. To print the current table, click Print.

A printable version is displayed.

  1. Before printing this version, peruse the pages to be printed and check the content, using the navigation toolbar or the zoom function.
  2. In the actions toolbar, click the Printer Settings button to select a printer and specify the printing options.

The setup dialog box for the printing opens up. For more information on this dialog box, see the documentation of your operating system.

  1. Specify the printing options, then click OK.
  2. In the actions toolbar, click the Print button.

The print task is launched.

You can also send the table by e-mail, click Notify in the actions toolbar.
  1. Click Close to go back to the setup page.
