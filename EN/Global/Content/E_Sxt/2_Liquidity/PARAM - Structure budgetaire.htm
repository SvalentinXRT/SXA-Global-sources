Budget Structures

The Budget Structure defines the budget codes that can appear in a given budget. It depends on the activity of your company.

In , to define the budget structure, you need to make a list of budget codes for the Operational Budget and another one for the Treasury Budget.

You can then associate different budget designers with this structure.

Accessing Setup Page for Budget Structures

In the pane, select the Budget Sheet Structure option in the folder of .

The setup page for budget sheets is displayed with the list of the set up structures in a table.

The table specifies the code and description of each structure.

 Creating Structures
  1. In the setup page for structures, click the Create button.
    The creation page for budget structures is displayed.

This page consists of two sections:

  • The header which includes the identification information of the structure (code and description)
  • The body of the page which enables you to associate budget codes with the structure It consists of two tabs, one for the Operational Budget, the other for the Treasury Budget.

For both tabs, the area on the left displays the existing budget codes and the area on the right displays the budget codes included in the structure.

  1. Enter a unique code, a description and potential comments for the structure.

The structure code is a unique identifier which cannot be modified later.

  1. In the Operational Budget Sheet tab, include the relevant budget codes in the structure of the Operational Budget Sheet, using the buttons between the areas.

View Buttons Description

Click this button to include all the existing budget codes in the structure.
Click this button to include the selected budget codes in the structure.
If you selected a budget category (for example Operating Receipts), all the budget codes of this category are included in the structure.
Click this button to include all the existing budget codes in the structure.
Click this button to remove the selected budget codes from the structure.
If you selected a budget category (for example Operating Receipts), all the budget codes of this category are removed from the structure.
  1. If you want to assign to a budget code a specific description for the selected structure, enter a new description instead of the default one, in the Description column in the Structure area.

The description of the budget code is modified for the selected structure. This description is to be displayed in your Budget Sheet.

The default description defined in the setup remains the same.
  1. If you want to move a budget code to another category than the default one, drag and drop it (i.e. click it and while holding the button down, move it to another location).

A yellow arrow is displayed during this action in order to show the target location for the budget type.

In the following example, the "RECDIVERS" (miscellaneous receipts) budget code is being relocated from the "Non-operating Receipts" category to the "Operating Receipts" category.

  1. To improve the budget sheet legibility, for example if there are too many budget codes, you may add blank lines in the structure.

For this purpose, insert the BLANKSPACE budget code.

You may insert more than one blank lines in the same structure.

These lines appear in the budget sheet and in the exercises comparison function, but they do not appear in reports.
  1. Click the Treasury Budget Sheet tab.

This tab is practically the same as the Operational Budget Sheet tab.

On the right of the page bottom, an additional button is displayed: Copy from OBS. It enables you to add the same budget codes as for the Operational Budget Sheet.

  1. Include the budget codes in the structure of the Treasury Budget Sheet, using the buttons between the areas (described above) and/or the Copy from OBS button.

For example, you can copy the budget codes for the structure of the Operational Budget Sheet, and add relevant system budget codes individually (subtotals, category totals and VAT).

  1. Click the following buttons, as needed:
  • Apply to save the setup of the structure and create another one
  • OK to save the setup of the structure and redisplay the list of the set up structures
  • Cancel to cancel the setup of the structure
 Duplicating Structures

You can create structures by duplicating the settings of existing structures.

  1. In the setup page for structures, select the structure to duplicate and click the Copy button in the actions bar.
  2. The creation page for structures completed with the settings of the duplicated structure, opens up. The code and description of the structure are the only ones not to be completed.

  1. In the Structure Code field, enter the new structure code and its description.

The structure code is a unique identifier which cannot be modified later.

You can modify the settings of the duplicated structure.

For more information on these fields, see the previous topic.

 Removing Budget Codes from Structures

You may remove budget codes from a structure previously set up, if both the following conditions are met:

  • None of the budget sheets associated with this structure has been closed.
  • No budget transaction has been saved for the budget codes on the budget sheets associated with this structure.

To remove budget codes from structures, proceed as follows:

  1. In the setup page for structures, select the structure to modify and click the Modify button in the actions bar.
  2. The modification page for structures is displayed.

For more information on the elements of this page, refer to the creation process for Budget Sheet Structures, described previously.

  1. In the OBS and TBS tabs, expand tree structures from the right column (which displays the budget codes included into the structure) using the + buttons.
  2. Click a budget code to select it, and click to remove it from the structure.
  3. Click OK to save your changes.

The management page for budget sheet structures appears.
