Consolidation at Company Group Level

The consolidation enables you to obtain a read-only global view of the budget at the level of groups or of units.

This function is available both in Budget Planning and Analysis modes.

If you manage one exercise per currency, this function is particularly useful to get a consolidated view of this budget in one currency. The consolidation currency is the display currency defined in the filter.

If you want to save your budget forecasts in different currencies, you first need to define a forecast rate for different currencies.
For more information on the forecast rates of currencies for multicurrency budgets, see the Currencies and Budget Rate topic.
Setting up Consolidation at Group Level
  1. In the tree view of budget exercises (in the left pane of the budget entry page), right-click an element and, from the contextual menu, select the Selection for Consolidation option.
  2. Press the Ctrl key, and, while still holding it down, click several budget versions to make a multiple selection.

If you manage one exercise per currency, you can select a version by currency to get a consolidated view of your budget in one currency. The consolidation currency is the display currency defined in the filter.

All the selected versions must have the same exercise periods. For example, you cannot obtain a consolidated view of one budget version for 2010 and of another one for 2011.
  1. Once you selected your versions, release the Ctrl key, right-click and, from the contextual menu, select the Consolidate Exercises option.

The selection page for budget sheet designers is displayed.

  1. In the Selection Designer field, choose a budget sheet designer using the button , then click OK to validate your selection.

The budget sheet is redisplayed with the consolidation of the selected budgets.

The amounts are displayed in view mode only.

 Setting up Consolidation at Unit Level
  1. In the tree view of budget exercises (in the left pane of the budget entry page), right-click an element and, from the contextual menu, select the Selection for Consolidation option.
  2. Press the Ctrl key, and, while still holding it down, click several units to make a multiple selection.

If you manage one exercise per currency, you can select a unit by currency to get a consolidated view of your budget in one currency. The consolidation currency is the display currency defined in the filter.

The exercise period must be the same for all the selected units and these units must belong to different companies. For example, you cannot obtain a consolidated view of one unit for 2010 and of another one for 2011.
  1. Once you selected your versions, release the Ctrl key, right-click and, from the contextual menu, select the Consolidate Exercises option.

The selection page for budget sheet designers is displayed.

  1. In the Selection Designer field, choose a budget sheet designer using the button , then click OK to validate your selection.

The budget sheet is redisplayed with the consolidation of the selected budgets.

The amounts are displayed in view mode only.

 Canceling Consolidated View

Right-click in the tree view of budget exercises (in the left pane of the budget entry page) and, from the contextual menu, select the Normal View of Exercise option.
