Request Monitoring

Two screens provide you with the view of validation requests:

  • The screen Validation Requests recaps all the requests.

  • The screen My Validation Requests recaps the requests of the connected user.

The display may be edited using the sorting and filtering functions.

The tasks specific to the requests are available from the actions menu.

  • View
  • Export to PDF
  • Export to Excel
  • Save
  • Reject Validation
  • Audit
