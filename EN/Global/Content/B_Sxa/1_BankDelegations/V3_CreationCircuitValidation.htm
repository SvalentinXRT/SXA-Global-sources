Creating Validation Workflow

now offers a validation cycle to strengthen the bank delegation process. This internal control system consists of various validation steps for delegations.

  1. From the Validation section of the Bank Delegations menu, select Validation Workflow.
  2. Click OK in the actions menu.

The Validation Workflow creation form appears.

  1. Complete the General Information section with the basics of the validation rule.

The Threshold amount field defines the amount from which the rule must be applied. When the threshold is null, the rule applies with all amounts.

Fields marked with an asterisk are mandatory.
  1. In the Companies section, define for which companies the rule must be applied.

  • Select General to apply the rule to all companies.
  • Select Specific one or more specific companies.

A new line is displayed.

  1. Click Select to open the selection window.

  1. Click the Accept button to validate your choice.

 

  1. Select the Validation Mode from the Authorization section.

  • The Level by level option sets priorities within the validation order: Level-2 users must validate before Level-1 users.
  • The Highest enough option cancels the prioritization: Level-3 users complete the validation (Validated status), regardless of any Level-1 and -2 Users' validation.

 

  1. Select one option from the Obsolete Requests section (History function).

  • The Delete button enables the limitation of data storage: no history available.
  • The Set done button enables you to keep all information related to validations and rejects. You can then view the validation history via the Audit checked function of a delegation.

 

  1. Click Add at the bottom of the form to define users level by level.

The section dedicated to Level 1 appears.

  1. From the Validation Rule section, select one validation option: Single or Double.
  2. Click Select and choose the Level-1 User(s) from the selection window.

  1. Click the Accept button to validate your choice.

 

  1. Repeat the actions of the previous step for all the validation levels, then click Save to save the new validation cycle.
