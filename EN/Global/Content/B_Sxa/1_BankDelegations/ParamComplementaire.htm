Additional Setup

Before using , you need to set up the following elements:

Identifier Types

For example, BIC.

 Account Categories

Here is the default standard list:

  • 0 = Undefined
  • 1 = Savings Account
  • 2 = Current Account
  • 3 = Security Account
  • 4 = Currency Account
  • 5 = Other

 Account Formats

For example, DEF-FR defines the default format of French Bank Accounts in the application.

 Branch Contacts

These contacts are associated with Branches. Provided you specified a main contact, you can associate several contacts with a same branch. These contacts are used for letters.

 Employees

The employees you have created may be defined as Delegators and/or Delegatees; additional documents can be attached.

In the Extension tab, and once you selected the option Use Employee for Extension, another option enables you to specify whether the employee is a Delegator or a Delegatee.

 Employee Groups

You can gather several employees into a group. One employee can belong to several Employee Groups.

 Companies

Specification of Delegators.

 Transaction Types

Operation Types are set up by the user. The Operation Type Descriptions entered upon creation, are then used for the letters.

Other examples:

  • Sign all cheques
  • Sign all authorizations
  • DC Issue
  • Sign transfer orders, etc.
  • Signature Type
 Signature Type

Transaction Types are set up by the user. The Signature Type Descriptions entered upon creation, are then used for the letters.

Other examples:

  • Acting separately
  • Acting jointly
  • 1st signatory
  • 2nd signatory, etc.
