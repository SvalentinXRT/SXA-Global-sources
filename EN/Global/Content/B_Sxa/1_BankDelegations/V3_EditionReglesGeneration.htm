Creating Generation Rules

In the set up form for the letter generation, three options enrich the General Information tab:

  • Add Cancelled Delegations
  • Delete Previous Drafts
  • Default template

Cancelled Delegations and Drafts

The Add Cancelled Delegations option enables the communication to the banks of the delegations that had been terminated since the last sent mail. The generated letters include both the current delegations and those that had been cancelled.

This option setup is a two-step process:

  1. Complete the Cancelled Delegations Data tab in the creation form for the letter.
  2. Select the Add Cancelled Delegations from the General Information tab.

The Delete Previous Drafts enables you to make sure you are working on a single version of the letter.

 

 Selecting Default Template

The Default Template section enables the selection of one template from the newly available ones.

All of them may be edited.

When the box in front of the template to modify is checked, the template actions appear in the action menu:

  • View
  • Change
  • Duplicate
  • Delete
  • Generate Letters by Rule
Duplicating letters is a simple and quick way to build up all your letters.
