FX Transaction Types

The default type of an FX transaction which was manually created is the "Initial" type.

When an FX transaction is carried forward or early settled, its type is modified as shown below. Its type is:

  • "Exercised" for an early settlement,
  • "Deferred" for a carry forward.

For a carry forward or an early settlement, two FX transactions are automatically generated:

  • one spot exchange transaction with the "Exercised" type for an early settlement or the "Deferred" type for a carry forward,
  • one forward exchange transaction with the "Exercising" type for an early settlement or the "Deferring" type for a carry forward,
