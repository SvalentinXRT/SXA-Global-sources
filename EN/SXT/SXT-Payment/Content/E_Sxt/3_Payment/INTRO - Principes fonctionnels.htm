Features

module was designed to enable you to fully manage your Payment Workflow using the following functions:

Transactions

This function enables you to create, manage and list by transaction type all transactions: Treasury Transfers (VTS), Domestic Transfers (VDOM), International Transfers (VINTL), etc.

 Setup

The setup functions enable you to adapt operational mode to the specificities of the specific Transfer Management of your company.

 Remittance

This function enables you to list and manage Remittances for all Payment Orders, whatever their transfer categories.

 Data Import and Export

Thoss functions enable you to import the payment orders created within another application, and to export transactions into other applications, which enhances your productivity.

 Validation Process

As the module has been designed to comply with the security rules for digital data exchange, this function enables you to automatically manage the various validation levels of transactions: validation, authorization and process, for optimal monitoring and task distribution.

 Analysis

With the statuses organization and the ledgers, module facilitates the detailed analysis of your payment requests and of the different transaction types (Domestic Transfers, Commercial Transfers, Treasury Transfers, etc.).

 Integration within Application Suite

uses the same flow basis and the same account balances than the module. Its data are completely consistent with the Cash data.
