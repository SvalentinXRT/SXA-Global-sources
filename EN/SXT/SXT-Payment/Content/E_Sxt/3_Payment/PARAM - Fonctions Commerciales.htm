Commercial Functions

The Commercial Functions are used to set up Bank Formats for Commercial Transfers.

You must initialize the codes linked to Commercial Functions, since you will be asked for them when entering Commercial Transfers. These codes follow the ISO standards. You can use the automatic initialization function and select the codes that you need.

Automatic Initialization

Upon the very first access to the Commercial Functions page, it is empty.

The Automatic Initialization button of the Commercial Functions menu enables the – partial or complete – retrieval of the functions predefined in the application through the Selection page.

the automatic initialization can then be performed at any time to reinitialize the function table, if, for instance, some functions were mistakenly modified or deleted.
Feeding Commercial Function Table
  1. To access this function, in the pane, select the Payment Elements option in the folder of the module.

The setup page for Receipts Elements is displayed. Click Commercial Function.

  1. Click the Automatic Initialization button to view the Selection window.

  1. Choose between the following selection methods:
  • Click All, if you plan to use all the preset functions. All of the selection boxes for Commercial Functions are ticked.
    You can still deselect functions by clicking their selection boxes.
  • If you only want some of the functions, select them one by one. To get more legibility, click first None to cancel all selections.
  1. Click OK to validate the operation, then Yes in the confirmation dialog box.

After clicking Yes, the following page is displayed:

The Code column lists the Commercial Functions Codes delivered with . These are the same as the EDIFACT codes. But you can still modify them.

The Descriptions column also presents the descritpions of the EDIFACT codes. But you can still modify them.

The EDIFACT column shows the EDIFACT-formatted codes used as references in the application. You cannot change the content of this comlumn, but you can associate its codes with those of the Codes column.

If you are using the automatic initialization function while standard commercial functions are listed in the Commercial Functions window, they will automatically be replaced by the original functions. This means that any modification made to the selected descriptions or to the selected EDIFACT codes, for example, will be lost. However, it will not affect the functions which you created.
Creating Commercial Functions
  1. To access this function, in the pane, select the Payment Elements option in the folder of the module.

The setup page for Receipts Elements is displayed. Click Commercial Function.

If you used the automatic initialization, the page displays the default functions.

  1. Click Create to open the creation page for Commercial Functions.

  1. Enter the function's ID in the Code field (limited to three alphanumerical characters).
  2. Enter a description in the Description field (limited to 35 alphanumercial characters).
  3. Click the arrow in the Code EDIFACT field to select from the dropdown list the standard that you want to associate with your commercial function.
  4. Click Apply to validate your action and go on with your creations, or click OK to save your new commercial function and go back to the list of the already created functions.
Commercial Function codes cannot be modified. Therefore, to change a code, you need to delete the function and to create it again under a new code.
Modifying Commercial Functions
  1. Select the Payment Elements option in the  folder of the module. Click Commercial Function.
  2. Select the function for which you want to modify the description or the EDIFACT code.
  3. Click the Change button, to open the Commercial Functions - Change page.
  4. Edit the type, then click OK to save it.
 Deleting Commercial Functions
  1. Select the Payment Elements option in the  folder of the module. Click Commercial Function.
  2. Select the function that you want to delete.
  3. Click Delete.
  4. Confirm your action by clicking Yes in the dialog box.
