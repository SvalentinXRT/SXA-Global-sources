File Paths

To establish the automatic storage of the remittances into a file ready to be transmitted to the banks, you need to specify the name and the path of the files to be sent.

Setting up Files Paths
  1. To access this function, in the pane, select the Payment Elements option in the folder of the module.

The setup page for Receipts Elements is displayed. Click File Paths.

  • Send: Address where the remittance files ready for the banks are stored. Their default path is: C:\ProgramData\SAGE\XRT_TREASURY\Send.
  • Receive: Address where the files returned by the banks are stored. Their default path is:

C:\ProgramData\SAGE\XRT_TREASURY\Receive

At the moment, this address is not used for .

  1. Enter all the paths in the corresponding fields or select successively the folders from Windows explorer. You can choose a directory via the button in each field.
  2. Click OK to save the paths for the send/receive directories, and to go back to the main page of .
