Processing Options

When you constitute the transfer remittances, you can also automate their processing.

  1. To access this function, from the pane, select the option in the folder of the module.

The setup page for Receipts Elements is displayed. Click the Process Type link in the Options section.

  • Generate: This option enables the generation of the file in a bank format.
  • Edit Slip: This option enables the edition of a remittance slip showing its details.
  • Edit Letter: This option enables the printing of a remittance advice letter to send to the provider. This letter informs the provider that the transfer was performed; it sums up all the transfer's characteristics (provider details, account, maturity, amount, description, etc.).
  • Archive: This option enables the archiving of a transfer assigned to a remittance, so that it is purgeable.
  1. Choose the processing options by selecting the corresponding boxes.
  2. Click OK to save the processing options and go back to the main page.
