Invoice Types

Invoice Types enable the definition of a type for each invoice associated with a commercial transfer.

Upon the very first access to the Invoice Types page, it is empty.

The Automatic Initialization button of the Invoice Types menu enables the – partial or complete – retrieval of the types predefined in the application through the Selection page.

The automatic initialization can then be performed at any time to reinitialize the function table, if, for instance, some functions were mistakenly modified or deleted. This command works exactly like the Automatic Initialization for Commercial Functions.

After the validation of your Invoice Types, the Invoice Types page has consequently been modified.

If you are using the Automatic Initialization function while standard invoice types are listed in the Invoice Types window, they will automatically be replaced by the original types. This means that any modification made to the selected descriptions or to the selected EDIFACT codes, for example, will be lost. However, it will not affect the types which you created.
Initializing Invoice Types
  1. To access this function, in the pane, select the Payment Elements option in the folder of the module.

The setup page for Receipts Elements is displayed. Click Invoice Type.

If you used the automatic initialization, the page displays the default types.

  1. Click Create to open the creation page for Invoice Types.

  1. Enter the type's ID in the Code field (limited to three alphanumerical characters).
  2. Enter a description in the Description field (limited to 35 alphanumercial characters).
  3. Click the arrow in the Code EDIFACT field to select from the dropdown list the standard that you want to associate with your invoice type.
  4. Click Apply to validate your action and go on with your creations, or click OK to save your new invoice type and go back to the list of the available types.
Invoice Type codes cannot be modified. Therefore, to change a code, you need to delete the type and to create it again under a new code.
Modifying Invoice Types
  1. Select the Payment Elements option in the  folder of the module. Click Invoice Type.
  2. Select the type for which you want to modify the description or the EDIFACT code.
  3. Click the Change button, to open the Invoice Types - Change page.
  4. Edit the type, then click OK to save it.
 Deleting Invoice Types
  1. Select the Payment Elements option in the folder of the module, then click Invoice Types.
  2. Select the invoice type that you want to delete.
  3. Click Delete.
  4. Confirm your action by clicking Yes in the dialog box.
