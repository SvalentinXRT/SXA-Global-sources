Cash Update Mode

The Cash Update Mode enables you to include transfers to the treasury forecasts. The Cash module is updated during the generation of the remittance to the bank.

Setting up the Cash Update Mode
  1. To access this function, in the pane, select the Payment Elements option in the folder of the module.

The setup page for Receipts Elements is displayed. Click Cash Update Mode.

The page for Cash Update Mode is displayed. You can view the already created Cash Update modes if exisiting, or create one.

  1. Select a transaction type from the dropdown list.
  2. Select a transaction code with the button of this field.
  3. Select the option By Order or By Remittance to define the Cash Update Mode. If you choose By Order, the application generates one forecast per transfer, with the amount of each transfer. If you choose By Remittance, the application generates one forecast per transfer remittance, with the global amount.
  4. Select a flow reference and description from the dropdown lists of the corresponding fields.
  5. Click Apply to validate your action and go on with your creations, or click OK to save your new mode and go back to the list of the already created modes.
You need to create as many update modes as transaction codes.
Changing the Cash Update Mode
  1. Select the Payment Elements option in the  folder of the module. Click Cash Update Mode.
  2. Select from the list the element you want to modify.
  3. Click Change. The edition page for the Cash Update Mode is displayed.
  4. Edit the Flow Reference and Flow Description fields, then click OK to save your action.
 Deleting the Cash Update Mode
  1. Select the Payment Elements option in the  folder of the module. Click Cash Update Mode.
  2. Select from the list the element you want to delete.
  3. Click Delete to delete the selected update mode.
  4. Confirm your action by clicking Yes in the confirmation dialog box.
