Data Export

This sectiontopic presents the export of Transfers created in and aimed at integration and exploitation in other systems.

Exporting consists in generating a flat file from the database.

The exported files can feed accounting systems or G/L entries generators for example.

The exported data comply with a predefined and fixed structure consisting of fields with fixed lengths and formats.

First of all, the structures of files to be exported must be set up in . You need to know the file structure required by the external system to set up properly this structure.
