Company

A company represents a unit of the budget evolution. Depending on your organization, a company can be a whole company, a department or a particular activity in your company.

For the setup of companies the following step is required: setting up the automated VAT calculation.

The company is defined in data repository. You can set them up via the module.

Accessing Setup Page for Companies
  1. In the pane, select the module from the accordion menu.
    The tasks of the module are displayed in this pane.
  2. Select the Companies option in the folder of this module.
    The setup page for companies is displayed with the list of the existing companies in a table.

The company's code is displayed in the second column of the table.

 Creating Companies
  1. In the setup page for companies, click the Create button.
    The Identification is displayed on the creation page. This tab enables you to save the standard information used for the unique identification of the company.

  1. In this tab, enter the code and description of the company.

View Fields Description

Field Description
Code Enter a unique code identifying the company.
It cannot be modified later.
Short Code This code is automatically completed from the defined standard code. If needed, it can be edited at creation time.
This code must be unique and cannot be changed later on.
Description Enter a description for the company
Additional You can add another description for your company
Currency Click the button to display the list of the set up currencies. In the contextual window, double-click the line to select it. The code and description of the selected currency are displayed in the Currency field.
  1. Click the Address tab.

  1. In the Country area, follow the process below to select the country of the company:
  1. Click the button to display the list of the set up countries.
  2. In the contextual window, double-click the line to select it.
    The code and name of the selected country are displayed in the Address tab.
  1. Click the Misc. tab.

This tab enables you to enter the settings for the automatic VAT calculation.

  1. In the section, complete the VAT Calculation, Direction and Priority areas.

View Fields Description

Field Description
VAT Calculation area
VAT Calculation Flag Select this option to enable the automatic VAT calculation in the Treasury Budget.
VAT is calculated with a default one-month rolling. In the Treasury Budget Sheet, the calculated VAT is displayed on two specific lines: the collected VAT for receipts (VATCOLL system code) and the retrievable VAT for disbursements (VATRETR system code).
If this option is not selected, VAT is not calculated in the Treasury Budget.
Prorate In this field, you can enter the percentage of retrievable VAT for paid VAT in case of expenses.
The default value for this field is 100%.
Priority area
Company Select this option so that the setup of the Direction field performed at the level of the company prevails over the setup performed at the level of the budget code.
Budget Code Select this option so that the setup of the Direction field performed at the level of the budget code prevails over the setup performed at the level of the company.
Direction area
Collection Select this option so that the transaction amount and date saved in the Treasury Budget are used as a basis for VAT calculation.
This option is mostly used by service providers.
Debit Select this option so that the transaction amount and date saved in the Operational Budget are used as a basis for VAT calculation.
The Debit option is selected by default.

To view examples of VAT calculation according to the direction, click here.

The other fields of this page are not relevant concerning the budget management.

  1. Click the following buttons, as needed:
  • Apply to save the setup of the company and create another one
  • OK to save the setup of the company and redisplay the list of the set up companies
  • Cancel to cancel the setup of the company
