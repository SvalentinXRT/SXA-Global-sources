Unit

Units are parts of the company. They are usually departments or a services within the company.

Since they correspond to sub-budgets, their setup is optional.

When you specify opening balances in the setup of the budget exercise, setting up units enables the determination of a percentage of opening balance to assign to each unit.

Setting up units is only possible if the company on which they depend, has already been set up.
Accessing Setup Page for Units

In the pane, click Units in the folder of the module.

The setup page for units is displayed with the list of the set up units in a table.

The table specifies the code and description for the unit, as well as the company on which it depends.

 Creating Units

To create units, you need to first create the company on which they will depend.

For more information on creating companies, see the Company topic.

  1. In the Actions bar in the setup page for units, click the Create button.
    The creation page for units is displayed.

  1. In the General Info area, enter a code and a description for the unit.
    It cannot be modified later.
  2. In the Code field of the Company area, click to select the company on which the unit depends.
    A contextual window displays the list of units.

  1. Double-click a line to select it.
    The code and description of the company are displayed in the Company area in the creation page for units.
  2. Click the following buttons, as needed:
  • Apply to save the setup of the unit and create another one
  • OK to save the setup of the unit and redisplay the list of the set up units
  • Cancel to cancel the setup of the unit
