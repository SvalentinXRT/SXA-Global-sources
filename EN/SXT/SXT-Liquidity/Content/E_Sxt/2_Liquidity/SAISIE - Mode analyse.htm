Budget Analysis

The Budget Analysis mode enables you to view and defer the discrepancies between the Treasury Budget and the flows saved in .

Analysis enables you to roll Non-actual Forecasts.

The budget sheet is displayed in analysis mode when you select this mode in the budget sheet filter.

The discrepancy between Forecasts and Actuals is null over the mixed budget period if, since the last actuals import, no cash flow was created nor modified in for the dates, company and budget codes involved in the budget exercise. For more information on the mixed budget, see the Importing Actuals procedure.

For more information on:

  • the setup options of the Analysis Mode (display of the budget and of the discrepancy with the flows of the module), see the Enabling Analysis Mode topic.
  • the options of the display filters (companies, division into periods, etc.), see the Filter Advanced Setup topic,
  • the rolling of non-actual forecasts, see the Rolling Non-Actual Forecasts topic.
  • the activation of the analysis mode, click here.

Showing or Hiding Discrepancies Showing or Hiding Treasury Budget or Flows saved in the Cash Module
  1. In the budget sheet, click the Change button in the Filters toolbar.
    The setup page for the selected filter is displayed.

  1. Click the View tab.

  1. In the BS views area, enable the Budget Analysis option.
    The fields of the area (specific to the budget analysis) are enabled.
  2. In the Type of Analysis area, enable the option:
  • Budget + Treasury to display the Treasury Budget and the flows saved in the module,
  • Budget to display only the Treasury Budget,
  • Treasury to display only the flows saved in the module,
  • Discrepancy to show only discrepancies and hide the Treasury Budget and the flows saved in the module.
  1. If you chose to view the Treasury Budget and the flows saved in the module, enable one of the following options in the Analysis Mode area:
  • Single Grid to display a single sheet (the Treasury Budget amounts are above the flows saved in the module),
  • Double Grid to display two sheets with the Treasury Budget and the flows saved in the module in two different sheets.

  1. In the Display Discrepancy area, enable the option:
  • Not show not to display the discrepancy (this option is disabled when the Discrepancy option is not selected in the Type of Analysis area),
  • Amount to display the discrepancy amount,
  • Percentage to display the discrepancy percentage.
  1. If you selected the double grid and the display of the discrepancy (enabled Amount or Percentage options in the Display Discrepancy area), enable one of the following options in the Discrepancy for Double Grid area:
  • Display in B grid to display the discrepancy in the Treasury Budget,
  • Display in T grid to display the discrepancy in the sheet of the flows saved in the module.
