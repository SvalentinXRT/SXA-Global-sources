General Toolbar

  • Disconnect: This button enables you to quit the application and go back to the home page.
  • View External Link: this option enables you to display the current view in a new web tab. The link which gives access to the view without using , is then displayed in the address bar. Other external applications can now access this view.
This extra option is part of the Mobile license.

  • Web View External Link: this option enables you to display the current view in a new web tab. The link which gives access to the view without using , is then displayed in the address bar. Other external applications can now access this view.
This extra option is part of the Web license.

  • Export: This submenu enables the current view to be exported into PDF, Excel or HTML. A message prompts you to select a location for the newly exported file.

  • Language Selection: provides multilingual support. Names for processes, views and fields may be viewed in other languages, provided the administrator selected this option. This function enables you to switch from one language to another.
  • Environment Selection: This menu enables the selection of the environment for the application operational data. The default environment name is displayed in the header. Environments are set up by the administrator and may correspond to various databases, or to various schemes within a same database.

  • Close: This button closes the toolbar.
In the upper part of the Command Center, the Filter list field enables you to enter a name or part of it (for processes, folders and views) to filter automatically the active list.
